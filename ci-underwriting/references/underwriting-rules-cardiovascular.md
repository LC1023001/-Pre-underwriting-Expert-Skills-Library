# 心血管疾病核保详细规则

## 适用险种：重大疾病险（CI）

---

## 1. 高血压核保详细规则

### 1.1 诊断标准（中国标准）
- 正常血压：收缩压 <120 mmHg 且舒张压 <80 mmHg
- 正常高值：收缩压 120-139 mmHg 或舒张压 80-89 mmHg
- 1级高血压：收缩压 140-159 mmHg 或舒张压 90-99 mmHg
- 2级高血压：收缩压 160-179 mmHg 或舒张压 100-109 mmHg
- 3级高血压：收缩压 ≥180 mmHg 或舒张压 ≥110 mmHg

### 1.2 核保评点表

| 分级 | 药物控制 | 靶器官损害 | NRS 评点 | 重疾险策略 |
|------|---------|----------|---------|---------|
| 正常高值 | 无需 | 无 | 0 | 标准体 |
| 1级，控制达标 | 是，达标 | 无 | +25 ~ +50 | 轻加费 |
| 1级，未药物干预 | 无 | 无 | +50 ~ +75 | 加费+高血压相关除外 |
| 1级，药物未达标 | 是，未达标 | 无 | +75 ~ +100 | 加费+除外 |
| 2级，控制良好 | 是 | 无 | +100 ~ +125 | 加费+心脑血管除外 |
| 2级，控制差 | — | 无 | +125 ~ +200 | 高额加费或拒保 |
| 任何级别 | — | 有（心/脑/肾/视网膜） | Dec | 拒保 |
| 3级高血压 | — | — | Dec | 拒保 |
| 继发性高血压（明确病因未处理） | — | — | Dec | 拒保 |

### 1.3 靶器官损害判定项目
- **心脏**：左室肥厚（心电图或超声心动图）、冠心病、心力衰竭
- **脑**：脑卒中/TIA 史、颈动脉内膜中层增厚 >1.0mm 或斑块
- **肾脏**：eGFR <60 mL/min、微量白蛋白尿（30-300 mg/24h）
- **视网膜**：高血压性视网膜病变 III-IV 级

### 1.4 收集资料要求
- 近3个月内血压测量记录（多次测量取均值）
- 心电图（判断左室肥厚、心律失常）
- 尿常规/尿微量白蛋白（肾脏损害筛查）
- 肌酐/eGFR（肾功能）
- 眼底检查（如有）

## 2. 冠心病核保详细规则

### 2.1 疾病分类
- 稳定型心绞痛
- 不稳定型心绞痛
- 急性心肌梗死（STEMI/NSTEMI）
- 陈旧性心肌梗死
- 经皮冠状动脉介入治疗（PCI）术后
- 冠状动脉旁路移植术（CABG）术后

### 2.2 核保评点表

| 情况 | 时间节点 | 心功能 | NRS 评点 | 重疾险策略 |
|------|---------|-------|---------|---------|
| 稳定型心绞痛，控制良好 | — | NYHA I | +75 ~ +100 | 加费+心血管除外 |
| 稳定型心绞痛，偶发 | — | NYHA I-II | +100 ~ +150 | 加费+除外 |
| 不稳定型心绞痛 | — | 任意 | Dec | 拒保 |
| 急性心肌梗死 | <2年 | 任意 | Dec | 拒保 |
| 急性心肌梗死 | 2~5年 | NYHA I | +150 ~ +200 | 延期评估/拒保 |
| 急性心肌梗死 | >5年 | NYHA I，EF≥50% | +100 ~ +150 | 高额加费+心血管除外 |
| PCI 术后 | <1年 | — | Dec | 拒保 |
| PCI 术后 | 1~3年 | NYHA I，无再狭窄 | +100 ~ +150 | 加费+心血管除外 |
| PCI 术后 | >3年 | NYHA I，EF≥50% | +75 ~ +100 | 加费+除外 |
| CABG 术后 | <2年 | — | Dec | 拒保 |
| CABG 术后 | 2~5年 | NYHA I，EF≥50% | +100 ~ +150 | 加费+心血管除外 |
| 心力衰竭（任何病因） | — | NYHA III-IV | Dec | 拒保 |

### 2.3 心功能 NYHA 分级
- I级：体力活动不受限，日常活动无症状
- II级：体力活动轻度受限，日常活动引起疲劳/心悸/气短
- III级：体力活动明显受限，轻于日常的活动引起症状
- IV级：不能进行任何体力活动，休息时也有症状

### 2.4 需要收集的资料
- 心电图（12导联）
- 超声心动图（射血分数 EF、室壁运动）
- 最近一次冠状动脉造影报告（如有）
- 运动负荷试验报告（如有）
- 心脏 CT 冠脉成像报告（如有）
- 相关用药情况（阿司匹林/他汀/β受体阻滞剂/ACEI等）

## 3. 心律失常核保详细规则

### 3.1 常见心律失常核保评点

| 心律失常类型 | 情况 | NRS 评点 | 重疾险策略 |
|------------|------|---------|---------|
| 窦性心动过缓（HR 50-60） | 无症状，运动员体质 | 0 | 标准体 |
| 窦性心动过缓（HR <50） | 需进一步排查 | 延期 | 评估后决定 |
| 窦性心动过速 | 器质性原因排除 | 0 ~ +25 | 轻加费或标体 |
| 室上性心动过速（SVT）| 偶发，无器质性心脏病 | +25 ~ +50 | 轻加费 |
| 室性早搏（偶发<1%） | 无器质性心脏病 | 0 | 标准体 |
| 室性早搏（频发≥1%） | 超声正常 | +25 ~ +50 | 轻加费 |
| 房性早搏 | 无症状 | 0 | 标准体 |
| 阵发性房颤 | 无血栓史，CHA₂DS₂-VASc≤1 | +50 ~ +75 | 加费+心房颤动相关除外 |
| 持续性/永久性房颤 | — | +100 ~ +200 | 加费+心脑血管除外 |
| 房颤合并血栓/栓塞史 | — | Dec | 拒保 |
| 完全性左束支传导阻滞 | 新发，原因未明 | 延期 | 排查器质性疾病后重评 |
| 完全性右束支传导阻滞 | 无器质性心脏病 | 0 ~ +25 | 标体或轻加费 |
| 预激综合征（WPW）| 无症状 | +25 ~ +50 | 加费 |
| 预激综合征（WPW）| 有症状，未手术 | +50 ~ +100 | 加费+心血管除外 |
| 预激综合征（WPW）| 射频消融术后，根治 | 0 | 标准体 |
| 室速/室颤史 | — | Dec | 拒保 |
| 起搏器植入 | 原因明确，功能正常 | +100 ~ Dec | 依原发病评估 |
| ICD 植入 | — | Dec | 拒保 |

## 4. 先天性心脏病核保规则

### 4.1 常见先天性心脏病分类与评点

| 疾病类型 | 手术状态 | 心功能 | NRS 评点 | 重疾险策略 |
|---------|---------|-------|---------|---------|
| 房间隔缺损（ASD）小型，≤5mm | 未手术 | 正常 | +25 ~ +50 | 轻加费 |
| ASD | 介入/手术根治，无残余分流 | 正常 | 0 ~ +25 | 标体或轻加费 |
| 室间隔缺损（VSD）小型（限制性） | 未手术 | 正常 | +25 ~ +50 | 加费 |
| VSD | 手术根治，无残余分流 | 正常 | 0 ~ +25 | 标体 |
| 动脉导管未闭（PDA）| 手术根治 | 正常 | 0 | 标准体 |
| 肺动脉瓣狭窄（轻度） | 未手术 | 正常 | +25 ~ +50 | 轻加费 |
| 复杂型先天性心脏病（法洛四联症等）| 任意 | — | Dec | 拒保 |
| 合并肺动脉高压（任何先心病）| — | — | Dec | 拒保 |

## 5. 心肌病核保规则

| 疾病 | 情况 | NRS 评点 | 策略 |
|------|------|---------|------|
| 扩张型心肌病 | EF <40% | Dec | 拒保 |
| 扩张型心肌病 | EF 40-50%，NYHA I-II | +150 ~ +200 | 高额加费或拒保 |
| 肥厚型心肌病（HCM） | 梗阻型 | Dec | 拒保 |
| 肥厚型心肌病（HCM） | 非梗阻型，无症状 | +100 ~ +150 | 加费+心血管除外 |
| 心肌炎（已痊愈 >1年，EF正常） | — | +25 ~ +50 | 轻加费 |
