## 二、核心发现：ST段改变是本案"一号核保障碍"

### 为什么ST段改变排名第一？

本案虽有乳腺BI-RADS 3类结节（多数公司可除外承保）和肺微小结节（2mm极小，多数公司可除外），但**ST段改变（II/III/aVF下移≥0.05mV）才是真正的核保杀手**：

1. ST段改变提示心肌可能缺血 → 重疾险核心风险（急性心梗位列28种必保重疾之首）
2. 伴随LDL-C边缘升高+全血低切粘度↑ → 动脉粥样硬化链条完整
3. CK 337显著升高 → 虽然CK-MB正常偏骨骼肌源，但在原因未明前是巨大的不确定性

### ST段改变在本案中的特殊之处

| 特征 | 本案情况 | 典型高风险 | 结论 |
|------|---------|-----------|------|
| 导联分布 | II/III/aVF三个下壁导联 | ≥3个导联 | 中等 |
| 下移幅度 | ≥0.05mV（边界值） | ≥0.1mV | 轻度 |
| T波 | 未见明显异常 | T波倒置 | 有利因素 |
| 伴随症状 | 心率80次/分，无胸痛主诉 | 胸痛/胸闷 | 有利因素 |
| 伴随风险因子 | LDL-C↑+CK↑ | 高血压/糖尿病 | 中度 |

> **核保逻辑**: 如果本案**仅有ST段改变无伴随因子**（LDL正常/CK正常），核保结论可能是标体或轻加费(+25)。但叠加LDL边缘升高和CK显著升高后，风险升级为+50~+75+心血管除外。

## 三、CK 337 U/L：最大的不确定性

### 为什么CK-MB正常是积极但不足以完全消除顾虑的信号？

| 指标 | 结果 | 参考值 | 解读 |
|------|:---:|:---:|------|
| CK | **337↑** | 26-140 | 2.4×ULN升高 |
| CK-MB | 13.0 | 0-25（正常） | **心肌同工酶正常** → 强烈提示骨骼肌源 |
| LDH | 160 | 103-227（正常） | 无组织损伤证据 |
| AST | 22 | 0-40（正常） | 无肝细胞/肌肉损伤证据 |
| ALT | 14 | 0-40（正常） | 肝功能正常 |

### CK升高的三种可能及核保影响

| 可能原因 | 概率 | 核保影响 |
|---------|:---:|---------|
| 近期剧烈运动（最常见） | 60% | 休息2周复查正常→标体 |
| 药物性（他汀/贝特） | 15% | 停药后复查正常→标体 |
| 亚临床心肌损伤（最严重） | 5% | 需心脏MRI+冠脉CTA→拒保/延期 |
| 肌病/甲减/其他 | 20% | 按具体诊断评估 |

> **关键不确定性**: 本案CK升高的原因未查明。如果复查正常 → 心血管评点可降至+35~+50。如果复查仍>300且无合理解释 → 建议延期至心内科明确诊断。

## 四、多系统微小结节：体质性倾向 vs 风险叠加

### 三系统结节特征

| 系统 | 分级 | 大小 | 数量 | 边界 | 血流 | 钙化 |
|------|:---:|:---:|:---:|:---:|:---:|:---:|
| 甲状腺 | TI-RADS 2 | 2×1mm | 单发 | 清 | 无 | 无 |
| 乳腺 | BI-RADS 3 | 5×3mm + 4×2mm | 多发 | 清 | 无 | 有(右乳钙化灶) |
| 肺 | Lung-RADS 2 | 2mm | 多发微小结节 | 清 | — | 有(肺尖陈旧钙化) |

### 核保评估三步骤

```
Step 1: 逐系统独立评估
  ├── 甲状腺 TI-RADS 2 (2mm) → NRS 0 → 标体
  ├── 乳腺 BI-RADS 3 (多发<1cm) → NRS +25 → 除外乳腺
  └── 肺 Lung-RADS 2 (2mm) → NRS +25 → 除外肺或观察

Step 2: 共病折叠
  ├── 甲状腺+乳腺 → 不同器官，但同属内分泌轴 → 取高（乳腺+25）
  ├── 甲状腺+肺 → 不同系统 → 独立评估
  └── 乳腺+肺 → 不同系统 → 独立评估（取高+25%）

Step 3: 多系统钙化灶
  ├── 乳腺钙化 → 与BI-RADS 3结节合并评估
  ├── 肺尖钙化+胸膜粘连 → 陈旧性病灶，标体
  └── 无活动性感染证据 → 不额外加费
```

### 关键结论

> **本案属于"体质性多系统微小结节"而非"癌症综合征"** — 三个系统均为良性分级+微小尺寸+无恶性特征。不应因结节数量多而升级核保结论。最终评点取高者（乳腺+25）再加肺（+25关联调整），综合结节评点≈+25~+50，除外乳腺+肺部。

## 五、技术沉淀：PaddleOCR v3+ API适配

### 3次迭代试错过程

| 迭代 | 尝试 | 错误 | 修复 |
|:---:|------|------|------|
| 1 | `PaddleOCR(use_angle_cls=True, lang='ch', show_log=False)` | `TypeError: unexpected keyword argument 'use_angle_cls'` | 删除已废弃参数 |
| 2 | `PaddleOCR(lang='ch')` + `ocr.ocr()` | `AttributeError: 'PaddleOCR' object has no attribute 'ocr'` | 改用 `ocr.predict()` |
| 3 | `ocr.predict()` 返回格式误解析 | 置信度全部为0% | 按 `rec_texts`/`rec_scores` 字段解析 |

### 最终可用代码

```python
from paddleocr import PaddleOCR

ocr = PaddleOCR(lang='ch')  # v3+ 最小初始化

def extract_text(result):
    """从predict()返回的OCRResult列表提取文字和置信度"""
    texts = []
    confs = []
    for item in result:
        for i, txt in enumerate(item.rec_texts):
            texts.append(txt)
            confs.append(item.rec_scores[i])
    return '\n'.join(texts), sum(confs)/len(confs) if confs else 0, len(texts)

# 使用
for img_path in page_files:
    result = ocr.predict(img_path)  # 不是 ocr.ocr()！
    text, conf, lines = extract_text(result)
```

## 八、反思与改进

### 本案的局限性

1. **年龄缺失** — 这是本案最大的信息缺口。35岁和55岁的核保结论差距可达"加费"vs"拒保"
2. **CK原因未明** — 在原因明确前，核保结论中存在+25的不确定性缓冲
3. **肺结节无历史对比** — 无法判断"稳定"还是"新发"，对核保影响评估有偏差

### 对未来评估的改进建议

1. **OCR后增加信息完整性检查** — 自动检测是否缺失年龄/性别等关键字段
2. **异常标记自动分级** — OCR后自动标记P0/P1/P2优先级，避免遗漏
3. **多次报告自动对比** — 如有历史报告，自动生成趋势对比表
