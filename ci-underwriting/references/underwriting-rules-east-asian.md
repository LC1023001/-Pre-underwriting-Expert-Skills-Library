# 东亚人群特异性核保规则

## 概述

东亚人群（中国大陆、港澳台、日本、韩国、东南亚华人）在某些疾病谱上与欧美人群存在显著差异，这要求重大疾病险的核保评估必须进行针对性调整。

以下规则综合了以下来源：
- 中再寿险电子核保手册（东亚人群专项数据）
- 瑞士再保险《中国人身险行业核保风控白皮书》（2021年）
- 世界卫生组织亚太区域流行病学数据
- 中国精算师协会《重大疾病经验发生率表（2020）》
- 日本、韩国、中国台湾精算学会人身险研究报告

---

## 一、乙型肝炎（HBV）— 东亚第一大核保差异点

### 1.1 流行病学背景
- 中国大陆 HBsAg 阳性率约 6-7%（约1亿感染者）
- 台湾、韩国约 3-4%
- 日本约 1%
- 欧美约 0.3%
- **结论**：东亚地区 HBV 相关风险是欧美标准核保手册的 3-10倍，必须专项管理

### 1.2 HBV 感染状态分类与核保评点

| HBV 感染状态 | 定义 | NRS 评点 | 重疾险策略 |
|------------|------|---------|---------|
| 自然获得免疫（抗-HBs+，HBsAg-）| 已康复或接种疫苗后免疫 | 0 | 标准体 |
| 接种后免疫（抗-HBs+） | 疫苗产生抗体 | 0 | 标准体 |
| 非活动性 HBsAg 携带者 | HBsAg+，DNA<2000 IU/mL，ALT持续正常，无肝纤维化 | +25~+50 | 轻加费+肝癌/肝脏相关重疾除外 |
| 慢性乙型肝炎（轻度活动）| HBsAg+，DNA 2000~20000，ALT轻度升高 | +50~+100 | 加费+肝脏相关重疾除外 |
| 慢性乙型肝炎（中重度活动）| HBsAg+，DNA>20000，ALT明显升高 | +100~+200 | 高额加费或拒保 |
| 乙肝相关肝纤维化（F1-F2）| 超声/弹性成像提示 | +100~+150 | 加费+肝相关重疾除外 |
| 乙肝相关肝纤维化（F3-F4/肝硬化）| — | Dec | 拒保 |
| 乙肝相关肝细胞癌 | — | Dec | 拒保 |

### 1.3 HBV 相关重要指标解读

| 指标 | 含义 | 核保关键值 |
|------|------|---------|
| HBsAg（表面抗原）| 感染标志 | 阳性则进入 HBV 核保流程 |
| 抗-HBs（表面抗体）| 免疫标志 | 阳性（+HBsAg阴性）= 标准体 |
| HBeAg（e抗原）| 病毒活跃复制标志 | 阳性提示高传染性、风险更高 |
| HBV DNA | 病毒载量 | <2000 IU/mL 为非活动，>20000 高风险 |
| ALT/AST | 肝细胞损伤 | ALT 持续 <40 U/L 且 HBV DNA 低是关键 |
| GGT | 胆道/肝损伤 | >60 U/L 需关注 |
| 白蛋白 | 肝合成功能 | <35g/L 提示肝功能受损 |
| 肝脏弹性成像（Fibroscan）| 肝硬度 | >9.5 kPa 提示纤维化 |
| AFP | 肝癌筛查 | >20 ng/mL 配合影像学评估 |

### 1.4 HBV 携带者额外材料要求
- 近3个月肝功能报告（ALT/AST/GGT/白蛋白）
- HBV DNA 定量（近12个月内）
- 肝脏超声报告（近12个月内）
- 如条件允许：肝脏弹性成像（Fibroscan）或 FIB-4 评分

### 1.5 FIB-4 评分（无创纤维化评分，HBV 核保辅助工具）
```
FIB-4 = 年龄(岁) × AST(U/L) ÷ [血小板(10⁹/L) × √ALT(U/L)]
```
- <1.30：低纤维化可能（F0-1），支持轻度加费策略
- 1.30~2.67：不确定，建议弹性成像确认
- >2.67：高纤维化可能（F3-4），评点上调

## 二、胃癌 — 东亚高发恶性肿瘤

### 2.1 流行病学背景
- 东亚地区（中国、日本、韩国）胃癌发病率占全球约75%
- 中国每年新发胃癌约43万例，死亡约28万例
- 主要风险因素：幽门螺杆菌（Hp）感染、腌制食品、高盐饮食、吸烟、家族史

### 2.2 核保风险调整
- 幽门螺杆菌（Hp）未根除：在标准 NRS 评点基础上 +25（东亚修正）
- 萎缩性胃炎（确诊）：+25~+50，消化道相关疾病除外
- 胃部腺瘤（已切除）：+50，消化道除外
- 一级亲属胃癌（<60岁确诊）：风险评点 +25（遗传修正）

## 三、鼻咽癌 — 华南/东南亚特有高发

### 3.1 流行病学背景
- 鼻咽癌发病率：华南（广东、广西）>东南亚华人 >台湾 >东北亚
- 广东省发病率约 30/10万，是欧美的 25倍以上
- 主要相关因素：EB病毒（EBV）感染、遗传、腌制食品（咸鱼）

### 3.2 EBV 抗体与核保
| EB 病毒抗体状态 | 核保处理 |
|-------------|---------|
| VCA-IgA 阳性（高滴度）| +25~+50，建议鼻咽镜检查，延期等结果 |
| EA-IgA 阳性 | +25~+50，同上 |
| VCA-IgA + EA-IgA 均阳性 | Dec/延期，高度怀疑，需鼻咽镜+活检 |
| 鼻咽部有占位，未活检 | Dec | 等病理结果 |

---

## 四、食管癌 — 中原/华北地区高发

### 4.1 流行病学背景
- 太行山区（河南、河北、山西交界）、山东、陕西等地高发
- 以食管鳞状细胞癌为主（与饮酒、烫食、腌制食物相关）
- 发病率是欧美的 5-10倍

### 4.2 核保注意事项
- 来自高发地区 + 家族史：整体风险评点轻度上调（+25）
- Barrett食管（已确诊）：+50，食管/消化道相关除外，建议定期内镜随访
- 食管重度异型增生：延期等内镜治疗（ESD）后重评

## 五、代谢性疾病 — 东亚 BMI 阈值差异

### 5.1 东亚 BMI 与代谢疾病关联差异

| BMI（东亚标准）| 代谢风险 | 核保说明 |
|-------------|---------|---------|
| <18.5 | 体重过低 | +25，排除消耗性疾病 |
| 18.5~22.9 | 理想 | 0，标准体 |
| 23.0~24.9 | 超重（预警）| 0~+25，关注血压/血糖/血脂 |
| 25.0~27.4 | 超重（轻度肥胖）| +25~+50，东亚人在此 BMI 段糖尿病风险显著上升 |
| 27.5~29.9 | 中度肥胖 | +50~+75 |
| 30.0~34.9 | 肥胖 | +75~+100 |
| ≥35.0 | 重度肥胖 | +150~Dec |

> **东亚修正依据**：WHO 2000年报告指出，亚太人群在 BMI 23-25 时已出现明显代谢风险，比欧美人群 BMI 25-30 更早。《中国居民肥胖诊断标准（WS/T 428-2013）》以 BMI 24 为超重界值，BMI 28 为肥胖界值。

### 5.2 代谢综合征东亚诊断标准（中华医学会内分泌学分会）

满足以下 3 项或以上即可诊断：
- 腰围：男性 ≥90 cm，女性 ≥85 cm
- 血压 ≥130/85 mmHg 或使用降压药
- 空腹血糖 ≥6.1 mmol/L 或使用降糖药
- 甘油三酯 ≥1.70 mmol/L
- HDL-C：男性 <1.04 mmol/L，女性 <1.30 mmol/L

代谢综合征核保评点：+50~+100（综合风险评估，关注心脑血管重疾风险）

## 六、地中海贫血 — 南方省份重要核保项目

### 6.1 地域分布
- 主要分布：广东、广西、湖南、四川、贵州、云南
- 广东人群携带率约 10-12%，广西约 20%
- 分型：α-地中海贫血、β-地中海贫血

### 6.2 核保评点表

| 基因型/表型 | 情况 | NRS 评点 | 重疾险策略 |
|---------|------|---------|---------|
| 静止型 α-地贫（单基因缺失，α-/αα）| 无贫血，血红蛋白正常 | 0 | 标准体 |
| 轻型 α-地贫（双基因缺失，α-/α-或--/αα）| 轻度小细胞低色素，Hb正常或轻度下降 | 0 ~ +25 | 标体或轻加费 |
| 中间型 α-地贫（血红蛋白 H 病，--/α-）| 中度贫血，偶需输血 | +50~+100 | 加费+贫血相关除外 |
| 重型 α-地贫（Hb Bart's 水肿胎儿）| 胎儿期死亡，不适用 | — | — |
| 轻型 β-地贫（杂合子）| Hb 轻度下降，MCV低 | 0 ~ +25 | 基本标体 |
| 中间型 β-地贫 | 中度贫血，偶需输血 | +50~+100 | 加费+贫血相关除外 |
| 重型 β-地贫（纯合子） | 严重贫血，依赖输血 | Dec | 拒保 |

## 七、吸烟与饮酒 — 行为风险东亚修正

### 7.1 吸烟
| 吸烟状态 | NRS 评点（额外）| 说明 |
|---------|-------------|------|
| 不吸烟 | 0 | 无调整 |
| 已戒烟 >5年 | 0 | 视同不吸烟 |
| 已戒烟 1~5年 | +25 | 轻度风险余留 |
| 吸烟者（每天 <10支）| +25~+50 | 依吸烟量 |
| 吸烟者（每天 10~20支）| +50~+100 | 中度风险 |
| 重度吸烟（>20支/天，>20包年）| +100~+150 | 高风险，肺部/心血管评点另算 |

### 7.2 饮酒（东亚乙醛脱氢酶缺乏特别说明）
> 约30-50%的东亚人因乙醛脱氢酶-2（ALDH2）基因突变而饮酒后面红心跳，此类人群饮酒相关食管癌、头颈部肿瘤风险显著升高。

| 饮酒状态 | NRS 评点（额外）|
|---------|-------------|
| 不饮酒 | 0 |
| 少量饮酒（<1单位/天）| 0 |
| 中度饮酒（1~2单位/天）| +25 |
| 重度饮酒（>2单位/天）| +50~+100 |
| 酒精依赖/酒精性肝病 | +100~Dec |

> 1单位酒精 ≈ 14g 纯酒精 ≈ 355mL 5%啤酒 / 150mL 12%葡萄酒 / 44mL 40%白酒

## 八、家族史风险修正表（东亚人群）

| 家族史情况 | 风险修正 | 重点关注疾病 |
|---------|--------|-----------|
| 一级亲属 <60岁确诊肝癌 | +25 | 乙肝、肝癌 |
| 一级亲属 <60岁确诊胃癌 | +25 | 胃癌（幽门螺杆菌家族聚集性） |
| 一级亲属 <60岁确诊结直肠癌，2人+ | +50 | 遗传性结直肠癌综合征（Lynch） |
| 一级亲属 <50岁确诊乳腺癌 | +25~+50 | 遗传性乳腺癌（BRCA1/2） |
| 一级亲属 2型糖尿病（2人+）| +25 | 糖尿病家族聚集性，东亚更明显 |
| 一级亲属早发脑卒中（<60岁）| +25~+50 | 脑血管病 |
| 一级亲属早发冠心病（男<55，女<65岁）| +25~+50 | 遗传性高脂血症 |

## 九、特殊人群核保注意事项

### 9.1 既往手术史
| 手术类型 | 核保处理 |
|---------|---------|
| 胆囊切除术（良性胆囊结石）| 0，标准体 |
| 脾切除术（外伤性）| +25，感染风险轻度上调 |
| 甲状腺切除（良性甲状腺肿）| 0 |
| 阑尾切除（无并发症）| 0 |
| 肾结石体外碎石/取石（单次）| 0 |
| 妇科良性肿瘤手术（已痊愈）| 0~+25 |
| 减肥手术（袖状胃/胃旁路）| +25~+50，依术后BMI和代谢改善情况 |

### 9.2 职业风险（东亚常见高危职业）
| 职业 | 主要风险 | 额外评点 |
|------|---------|---------|
| 矿工（煤矿/石英矿）| 尘肺、肺癌 | +50~+100 |
| 电焊/油漆工 | 肺部疾病、溶剂中毒 | +25~+50 |
| 放射科医技人员 | 辐射暴露（现代防护标准下影响小）| +25 |
| 高温/重体力劳动 | 心血管应激 | +25 |
| 夜班/倒班工作者 | 代谢综合征风险 | 观察，非直接加费 |
