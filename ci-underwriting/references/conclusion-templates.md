# 核保结论模板库

## 模板一：标准体承保（标体）

```
===========================================
【重大疾病险核保评估意见书】
===========================================

一、被保险人基本信息
  · 姓名：[如已知]
  · 年龄/性别：XX岁 / 男性 or 女性
  · 拟投保险种：重大疾病险（终身型/定期型）
  · 申请保额：XX万元

二、医学资料摘要
  · 已提供资料：年度健康体检报告（日期：XXXX-XX-XX）
  · 关键发现：
    1. [各项指标名称] — [数值/结果]：正常范围内
    2. 未见明显异常
    
三、核保风险评估

  所有健康指标均在正常范围内，未发现影响重大疾病险承保的医学异常。
  综合 NRS 评点：0分（标准体）

四、核保结论
  ┌────────────────────────────────┐
  │        核保结论：标准体承保          │
  └────────────────────────────────┘

  · 保费调整：标准费率，无额外加费
  · 除外责任：无
  · 特别备注：无

五、建议
  · 建议每年定期进行健康体检
  · 维持健康生活方式（适度运动、均衡饮食）
  
===========================================
本意见为辅助评估，最终承保以保险公司核保部门决定为准。
===========================================
```

## 模板三：除外责任承保

```
===========================================
【重大疾病险核保评估意见书】
===========================================

一、被保险人基本信息
  · 年龄/性别：XX岁 / [性别]
  · 拟投保险种：重大疾病险
  · 申请保额：XX万元

二、医学资料摘要
  · 已提供资料：[资料类型及日期]
  · 关键异常发现：
    1. [核心异常项目]：[具体数值/描述]

三、核保风险评估
  
  [核心异常项目] 风险评估：NRS 评点 +XX
  
  鉴于该异常与特定重大疾病存在明确关联，建议采用"除外责任"方式承保，
  即保单保障范围中不涵盖与该异常相关的重大疾病理赔。

四、核保结论
  ┌────────────────────────────────┐
  │        核保结论：除外责任承保         │
  └────────────────────────────────┘

  · 保费调整：[标准费率 / 额外加费 XX%]
  · 除外责任：
    - 除外"[具体除外疾病名称]"及其相关并发症的保障
    - 具体除外范围请参照保单条款
  · 除外期限：[永久性除外 / X年后可申请解除]

五、建议与说明
  · [具体健康管理建议]
  · X年后，若[除外相关指标]恢复正常，可携带检查报告申请重新核保，
    届时核保部门可根据新情况评估是否解除除外责任。
  
===========================================
```

## 模板六：拒保

```
===========================================
【重大疾病险核保评估意见书】

    [描述导致拒保的核心情况]

三、核保风险评估

  经评估，被保险人当前健康状况所对应的核保风险评点超过重大疾病险可承保阈值，
  具体原因如下：
  
  [具体原因，如：
  - 确诊[疾病名称]，属于重大疾病险绝对拒保情况
  - 综合 NRS 评点达 +XXX，超过重大疾病险可承保范围（通常 ≤+200）
  - [具体疾病] 处于活动期/治疗期]

四、核保结论
  ┌────────────────────────────────┐
  │           核保结论：拒保               │
  └────────────────────────────────┘

  · 当前重大疾病险无法承保
  
五、其他建议
  · 目前可考虑的保险替代方案：
    - 意外险：通常不受健康状况影响，可正常投保
    - 普通医疗险（部分产品对该情况核保较宽松，建议咨询）
  · [如有康复可能]：若[具体条件]（如治疗完成、康复X年、检查指标恢复正常），
    建议届时重新申请核保，届时将重新评估承保可能性。
  · [健康管理建议]
  
  注：拒保结论基于当前提供的医学资料及核保规则，不代表对其健康状况的医学判断。
  
===========================================
本意见为辅助评估，最终承保以保险公司核保部门决定为准。
===========================================
```

## 常用除外责任标准措辞参考

| 除外范围 | 标准措辞示例 |
|---------|-----------|
| 肝脏相关 | 除外因乙型肝炎（包括HBV携带者状态）引起或导致的肝脏疾病（包括但不限于肝硬化、肝细胞癌）引发的保险金给付责任 |
| 高血压相关 | 除外因原发性高血压引起或导致的心、脑、肾等靶器官损害（包括但不限于高血压性心脏病、脑中风）引发的保险金给付责任 |
| 糖尿病相关 | 除外因糖尿病及其并发症（包括但不限于糖尿病性肾病、视网膜病变、神经病变、血管病变）引发的保险金给付责任 |
| 心血管相关 | 除外因冠心病（包括但不限于心绞痛、急性心肌梗死）引发的保险金给付责任 |
| 甲状腺相关 | 除外因甲状腺疾病（包括甲状腺结节、甲状腺肿瘤）引发的保险金给付责任 |
| 肺部相关 | 除外因慢性阻塞性肺病（COPD）/哮喘/支气管扩张引发的保险金给付责任 |
| 肿瘤除外 | 除外因[具体肿瘤类型]引发的恶性肿瘤保险金给付责任 |
| 消化道相关 | 除外因消化道息肉/溃疡性结肠炎/克罗恩病引发的保险金给付责任 |

## 附：核保结论快速参考矩阵

| NRS综合评点 | 除外情况 | 建议核保结论 |
|----------|---------|-----------|
| 0 | 无 | 标准体承保 |
| +25~+50 | 无 | 轻加费承保（+25%~+50%） |
| +50~+75 | 1项 | 加费或除外（视具体情况） |
| +75~+100 | 无 | 中度加费（+75%~+100%，即双倍保费） |
| +75~+100 | 1~2项 | 加费+除外组合 |
| +100~+150 | 无 | 高额加费（接近极限）或拒保 |
| +100~+150 | 1~3项 | 加费+除外，或延期 |
| >+150 | 无除外可解决 | 拒保 |
| >+200 | — | 拒保（进入经验核保范围） |
| 病情不稳定 | — | 延期（PP） |
| 活动期恶性肿瘤/严重系统疾病 | — | 拒保（Dec） |
