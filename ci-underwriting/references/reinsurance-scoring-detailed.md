## 一、Munich Re — MIRA 系统评分详表

| 情况 | MIRA Table | 基础评点 | 说明 |
|------|---------------|-------:|------|
| 室早负担 <1% | Table 1 | +0~+25 | 几乎标准体 |
| 室早负担 1-3% | Table 2 | +25~+50 | 轻度加费 |
| 室早负担 3-5% | Table 2-3 | +50~+75 | 中度加费 |
| 室早负担 5-10% | Table 3 | +75~+100 | 高额加费 |
| 室早负担 >10% | Table 4 | +100~+200 | 高加费或拒保 |
| 房颤（阵发性） | Table 3-4 | +50~+100 | 需抗凝评估 |
| 房颤（持续性） | Table 4-5 | +100~+200 | 高加费 |
| 预激综合征（无症状） | Table 2 | +25~+50 | 射频消融后评点↓ |

**特殊考量**：三联律 >50阵/24h → 评点 +25；成对室早 >0 → 评点 +25。

## 二、Swiss Re — 心血管风险评分表

### 2.1 室性心律失常评分

| PVC 负担 | 心脏超声 | Swiss Re Rating | 加费系数 |
|-----------|---------|-----------------|--------:|
| <1% | 正常 | Class 1 | 0% |
| 1-3% | 正常 | Class 2 | +25~+50% |
| 3-5% | 正常 | Class 2-3 | +50~+75% |
| 5-10% | 正常 | Class 3 | +75~+100% |
| >10% | 正常 | Class 3-4 | +100~+200% |
| 任意 | 异常 | Class 4-5 | +200%~Dec |

**关键阈值**：Swiss Re 以 **5%** 为重要分界点——>5% 自动进入 Class 3 以上。

### 2.2 房颤（Atrial Fibrillation）评分

结合 CHA₂DS₂-VASc 评分：

| CHA₂DS₂-VASc | Swiss Re 立场 | 加费系数 |
|--------------------|---------------|--------:|
| 0分（男性） | Class 2 | +25~+50% |
| 1-2分 | Class 3 | +50~+100% |
| ≥3分 | Class 4 | +100~+200% |

## 三、Gen Re — 再保核保指引速查

### 3.1 室早（PVC）核保指引

| PVC 负担 | 器质性心脏病 | Gen Re 分类 | 评点 | 观察期 |
|-----------|---------------|---------------|-------:|--------:|
| <1% | 无 | Low risk | +0~+25 | 无 |
| 1-5% | 无 | Moderate risk | +25~+50 | 建议1年后复查 |
| 3-5% | 无 | Moderate-High | +50~+100 | 建议6个月后复查 |
| 5-10% | 无 | High risk | +100~+150 | 建议3个月后复查 |
| >10% | 无 | Very high | +200%~Dec | 延期 |
| 任意 | 有 | Very high | Dec | 拒保 |

> **Gen Re 核心立场**：PVC 负担 >5% → 建议 **除外心血管 + 加费 50-100%**。

### 3.2 甲状腺癌术后指引

| TNM | 术后年限 | Gen Re 立场 | 评点 |
|-----|---------|---------------|-------:| 
| T1aN0 | >3年 | 标准体 | +0~+25 |
| T1bN0 | >5年 | 标准体或除外 | +0~+25 |
| T1-3N1a | >5年 | 除外+加费 | +50~+75 |
| **T1-3N1b** | **>5年** | **除外+加费 50-100%** | **+50~+100** |
| T1-3N1b | 2-5年 | 延期或高额加费 | +100~+200 |

## 四、Hannover Re — 自动化核保规则

### 4.1 心律失常自动化规则

| 情况 | 自动化决策 | 说明 |
|------|---------|------|
| 室早 <1% + 心脏超声正常 | 自动通过（标准体） | 无需人工审核 |
| 室早 1-5% | 自动加费 +25~+50% | 智能核保可直接出单 |
| 室早 >5% | 自动转人工核保 | 需提供心脏超声 + 运动试验 |
| NSVT（非持续性室速） | 自动延期/拒保 | 需心脏 MRI 排除瘢痕 |
| 房颤（阵发性） | 自动加费 +50~+100% | 需抗凝评估 |

### 4.2 甲状腺癌术后自动化规则

| 情况 | 自动化决策 |
|------|---------|
| T1aN0 术后 >3年 | 自动通过（标准体） |
| T1bN0 术后 >5年 | 自动通过或除外甲状腺 |
| T1-3N1a 术后 >5年 | 自动除外+加费 |
| T1-3N1b 术后 <3年 | 自动延期 |
| T1-3N1b 术后 3-5年 | 人工审核，通常除外+加费 |

## 五、SCOR — 亚洲市场经验

### 5.1 亚洲人群心律失常核保经验

| PVC 负担 | 亚洲市场尺度 | 加费范围 |
|-----------|---------------|--------:|
| <1% | 宽松 | +0~+25% |
| 1-3% | 中等 | +25~+50% |
| 3-5% | 中严 | +50~+75% |
| 5-10% | 严格 | +75~+100% |
| >10% | 拒保 | Dec |

> **SCOR 亚洲经验**：亚洲人群（尤其中国）器质性心脏病发病率低于欧美，但 **心律失常就医依从性差**（不就诊 = 未"确诊" → 智能核保可能按标体承保，存在逆选择风险）。

### 5.2 亚洲人群甲状腺癌术后经验

N1b 在亚洲人群中复发率约 15-20%（高于欧美 10-15%），再保对此更谨慎。

## 七、再保分保操作实务

### 7.1 高保额触发再保体检

| 险种 | 保额阈值 | 再保要求 |
|------|---------|---------|
| 重疾险 | >100万 | 再保体检（Holter + 心脏超声 + 运动试验） |
| 定期寿险 | >200万 | 再保体检（Holter + 心脏超声） |
| 医疗险 | >500万 | 再保体检（全面） |

### 7.2 再保拒保/加费情形

| 情形 | 再保立场 | 操作建议 |
|------|---------|---------|
| PVC 负担 >10% + 心脏超声异常 | 拒保分保 | 建议直接拒保 |
| PVC 负担 5-10% + Holter 三联律 >100阵 | 高加费分保（+200%） | 建议高额加费 |
| 甲状腺癌 T1bN1b 术后 <3年 | 延期分保 | 建议延期至术后3年 |
| 肺结节 Lung-RADS 4A类 | 延期分保 | 建议穿刺后重评 |
