# 智能核保产品规则库（v2.8新增）

> 沉淀来源：陶玉峰案例（室早）+ 张瑜案例（HPV/先心）+ 戴薇案例（PTC术后）+ 王烨佳案例（射频术后/桥本/IMT）+ 复兴联合/好医保/众安产品实测经验。

---

## 一、复兴联合健康（智能核保代表产品）

### 1.1 重疾险智能核保规则（复兴联合「康爱无忧」等）

| 健康异常 | 智能问卷问题 | 核保结论 | 说明 |
|---------|---------------|---------|------|
| 室早/心律失常 | "是否被诊断为心律失常？是否行心脏射频消融/起搏器植入？" | **仅问"诊断"** → 未确诊者可能**标体** | 智能规则通常不细分 Holter 负担，仅问"是否被诊断" |
| 肺结节 | "是否有肺结节/肺磨玻璃结节？结节是否>8mm？是否术后病理良性？" | <6mm → 标体或轻微除外；6-10mm → 除外肺部；>10mm → 延期 | 不区分 Lung-RADS，仅问大小 |
| 甲状腺结节 | "是否 TI-RADS ≥4级？是否术后病理良性？" | TI-RADS 1-3级 → 标体；4级 → 延期/除外 | 射频术后可能无法通过智能核保 |
| 乙肝病毒携带 | "乙肝表面抗原是否阳性？HBV DNA 是否阳性？ALT 是否>2×ULN？" | 表面抗原(+) + DNA(-) + ALT正常 → 加费25-50%；活动期 → 延期 | 智能规则比人工宽松 |
| 肾结石 | "是否有肾结石？是否引起肾积水/肾功能异常？" | 无积水 + 肾功能正常 → 标体；有积水 → 除外泌尿 | 不细分结石大小 |
| 前列腺钙化 | **通常不问** | 标体 | 智能问卷无此项 |

> **关键经验**：智能核保对 PTC 术后（尤其 N1b）可能**比传统人工核保更宽松**，因其规则仅区分"是否癌症史+术后年限"，不细分 N0/N1a/N1b。

### 1.2 医疗险智能核保规则（复兴联合「医爱无忧」等）

| 健康异常 | 智能问卷问题 | 核保结论 | 说明 |
|---------|---------------|---------|------|
| 室早/心律失常 | 同重疾险 | 通常**除外心血管** | 医疗险比重疾险严 |
| 肺结节 | 同重疾险 | <6mm → 除外肺部；≥6mm → 延期 | 医疗险几乎一定除外 |
| 甲状腺结节/术后 | "是否术后 >3年？病理是否良性？" | 术后≥3年 + 病理良性 → 除外甲状腺；<3年 → 延期 |  |
| 肾结石 | 同重疾险 | 无积水 → 标体或除外泌尿 |  |

## 二、好医保（人保健康，互联网重疾险代表）

### 2.1 好医保·长期医疗（20年保证续保）

| 健康异常 | 健康告知问题 | 核保结论 | 说明 |
|---------|---------------|---------|------|
| 心律失常 | "是否有心律失常（不含窦性心动过速/过缓）？是否行心脏射频消融？" | 窦性心律 + 未行消融 → **标体可能** | 仅排除"被诊断的心律失常" |
| 室早 | **通常不专门问** | 若未诊断为"心律失常" → 标体 | Holter 异常但未就医 = 未告知也不违规（但道德风险） |
| 肺结节 | "是否有肺结节/肿块/磨玻璃结节？" | 有 → 人工核保或除外肺部 | 智能核保可能无法直接通过 |
| 肝炎/乙肝 | "是否有肝炎/肝硬化？" | 乙肝携带 + 肝功能正常 → 除外肝病或加费 |  |

> **好医保特殊规则**：部分版本有"智能核保自动通过"功能——填写健康告知后系统自动判断是否承保。

### 2.2 好医保·重疾险（互联网重疾）

规则与复兴联合类似，但对甲状腺结节/乳腺结节更宽松（BI-RADS 3级有时标体）。

## 三、众安保险（众安在线，智能核保技术领先）

### 3.1 众安尊享 e 生（医疗险）

| 健康异常 | 智能问卷 | 核保结论 | 说明 |
|---------|---------|---------|------|
| 心律失常/室早 | 有专门"心悸/胸闷"问卷 | 提供 Holter 正常 → 可能标体；Holter 异常 → 除外心血管 | 众安对心律失常有专门问卷 |
| 肺结节 | "肺结节大小/性质/随访" | <6mm + 稳定2年 → 除外肺部；≥6mm → 延期 |  |
| 乳腺结节 | "BI-RADS 分级" | BI-RADS 1-2级 → 标体；3级 → 除外乳腺 |  |

### 3.2 众安重疾险（守护宝等）

与众安医疗险类似，但对甲状腺结节术后 N1b 可能有专门问卷（需实测）。

## 四、智能核保 vs 人工核保——策略选择矩阵

| 投保人情况 | 推荐渠道 | 理由 |
|---------------|---------|------|
| PTC 术后 N0，术后≥3年 | **智能核保**（复兴联合/好医保） | 智能规则不细分 N 分期 → 可能仅除外 |
| PTC 术后 N1b，术后 2-5年 | **人工预核保**（经纪人渠道） | 智能可能直接拒保；人工可附资料争取 |
| 室早 <1%，心脏超声正常 | **智能核保** | 智能可能标体（未问及 Holter） |
| 室早 3-5%，Holter 证实 | **人工核保** | 智能可能拒保；人工可争取加费 |
| 肺结节 <6mm，稳定≥2年 | **智能核保** | 多数公司标体或仅除外 |
| 多异常（室早+肺结节+肾结石） | **人工核保** | 智能通常直接拒保；人工可组合评估 |

## 五、智能核保「未告知」的法律风险

> ⚠️ **重要提示**：智能核保虽未专门询问某些异常（如 Holter 发现的室早），但《保险法》第 16 条规定"投保人故意或因重大过失未履行如实告知义务，足以影响保险人决定是否同意承保或提高保险费率的，保险人有权解除合同"。

**实操建议**：
1. 智能核保未问及的异常 → 可不主动告知（不违规）
2. 但**如果**保险公司要求"填写详细病史"或"提供体检报告" → 必须告知
3. 室早 3.7% 负担 + 三联律 → **建议主动告知并提供 Holter 报告**，避免未来理赔纠纷
