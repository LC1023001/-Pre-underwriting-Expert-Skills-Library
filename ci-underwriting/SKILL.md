# 重大疾病险智能核保系统

## 系统定位

本 Skill 依据全球前十大再保险公司核保手册（慕尼黑再保险 MIRA/NRS 系统、瑞士再保险、汉诺威再保险、中再寿险等）与中国大陆主要人身险公司核保规范（平安、太平洋、中国人寿、泰康、友邦等），结合《重大疾病保险的疾病定义使用规范（2020年修订版）》，专为东亚人群（中国大陆、港澳台、日韩）健康风险特征优化，提供科学、合理、可落地的重大疾病险核保评估。

> 重要声明：本系统输出为辅助评估意见，最终核保决定须由持牌核保人员依据保险公司具体核保规程作出。

## 核保工作流程

当用户提供医学资料请求核保评估时，按以下步骤执行：

### 步骤一：信息收集与整理

收集并整理以下维度的信息：

**基本信息**
- 年龄、性别（不同年龄段风险权重不同）
- 拟投保险种（重疾险保额、保险期间）
- 家族病史（尤其一级亲属恶性肿瘤、心血管、糖尿病史）

**医学资料来源**
- 体检报告（年度健康体检）
- 医院病历（门诊/住院记录）
- 检验报告（血常规、生化、肿瘤标志物等）
- 影像报告（超声、CT、MRI、X线）
- 病理报告（活检结果）
- 手术记录
- 出院小结

### 步骤二：NRS 数字评点评估

采用改良 NRS（Numerical Rating System）数字评点法，参考慕再核保手册：

**评点基准**：以标准体死亡率/发病率为 100%，额外风险用百分点表示。

**重大疾病险评点决策标准**：
- **0点（标准体）** → 标准保费承保（标体）
- **+25 ~ +75点** → 轻度加费承保（额外保费约 +25%~+75%）
- **+100点** → 中度加费承保（额外保费 +100%，即双倍保费）
- **+100点 + 除外责任** → 组合条件承保
- **+200点以上** → 高度加费或拒保（重疾险通常不超过3项除外）
- **延期（PP）** → 病情不稳定，需观察后重新评估
- **拒保（Dec）** → 风险过高，无法承保

> 重疾险核保原则：重大疾病险的评点通常为 0（标准体）或 +100%（双倍费率），高风险情况优先采用"除外责任"而非大额加费，且建议除外责任不超过3项。

### 步骤三：东亚人群风险修正

在 NRS 基础上，叠加以下东亚人群特异性风险调整：

**高发疾病上调权重**（与欧美标准相比）：
- 乙型肝炎（HBV）携带/慢性乙肝：风险权重 ×1.5
- 肝细胞癌（与 HBV 高度相关）：风险权重 ×1.8
- 鼻咽癌（东南亚及华南人群高发）：风险权重 ×2.0
- 胃癌（与 Hp 感染、饮食习惯相关）：风险权重 ×1.5
- 食管癌（华北/中原地区高发）：风险权重 ×1.4
- 2型糖尿病（亚裔 BMI 阈值更低）：BMI≥23 即启动代谢风险评估
- 代谢综合征（脂肪肝、高脂血症）：综合评分
- 地中海贫血（南方省份）：需关注基因携带状态

**风险下调（相对欧美人群）**：
- 前列腺癌：东亚发病率低，风险权重 ×0.5
- 黑色素瘤：东亚发病率低，评估可适当宽松
- 心血管事件（冠心病）：50岁以下东亚男性相对风险低于欧美，但不得忽视高血压和脑卒中风险

### 步骤四：核保结论输出

生成格式化核保意见书（见"输出模板"章节）。

## 核保评估规则库

### 一、基础生命指标评估

**BMI（东亚适用标准）**

| BMI 范围 | 评点 | 说明 |
|---------|------|------|
| 18.5 ~ 22.9 | 0（标准） | 东亚理想体重 |
| 23.0 ~ 24.9 | 0（标准，观察） | 超重预警，关注血压血糖 |
| 25.0 ~ 27.4 | +25 | 轻度超重，关注代谢指标 |
| 27.5 ~ 29.9 | +50 | 中度超重，需关注合并症 |
| 30.0 ~ 34.9 | +100 | 肥胖，建议加费或组合除外 |
| ≥35.0 | +150 ~ Dec | 重度肥胖，高风险 |
| <18.5（体重过低） | +25 | 需排除慢性消耗性疾病 |

> 东亚修正：中国居民 BMI≥24 即定义为超重，BMI≥28 为肥胖，比欧美标准低。

**血压（中国标准）**

| 血压状态 | 评点 | 核保策略 |
|---------|------|---------|
| <120/80 mmHg | 0 | 标准体 |
| 120-139/80-89（高正常） | 0 ~ +25 | 观察，关注是否升级 |
| 1级高血压（140-159/90-99），药物控制达标 | +25 ~ +75 | 加费，除外高血压相关重疾 |
| 1级高血压，未用药或控制不佳 | +75 ~ +100 | 加费+除外 |
| 2级高血压（160-179/100-109） | +100 ~ Dec | 加费+除外或延期 |
| 3级高血压（≥180/110） | Dec | 拒保 |
| 高血压合并靶器官损害（心/脑/肾） | Dec | 拒保 |

#### 一过性高血压/白大衣高血压核保专项（v3.0新增）

> **沉淀来源**: 曹治家案例（2017年165/99→2021年130/81→当前130/90，无服药史，无靶器官损害）+ 临床经验。

**一、一过性高血压的核保判断路径**

```
诊室血压升高（≥140/90）→ 鉴别：
├── 仅单次/少数几次升高，后续多次正常
│   ├── 无服药
│   ├── 无靶器官损害（心超/肾功/眼底正常）
│   ├── 无代谢异常（血糖/血脂正常）
│   └── → 白大衣高血压（WCH）或一过性升高
│       → 核保：**标准体**（多数公司）
│
├── 间歇性升高，有时正常有时偏高
│   ├── 家庭自测血压偏高（≥135/85）
│   ├── 24h动态血压监测（ABPM）异常
│   └── → 隐匿性高血压或早期高血压
│       → 核保：按高血压实际分级处理
│
└── 持续升高，需药物治疗
    → 确诊高血压
    → 按高血压分级+控制情况核保（见上表）
```

**二、一过性高血压核保决策矩阵**

| 情况 | 血压值 | 持续时间 | 服药 | 靶器官损害 | NRS评点 | 核保策略 |
|------|--------|---------|:---:|:---------:|:-------:|---------|
| **白大衣高血压** | ≥140/90（仅诊室） | 偶发 | 无 | 无 | **0** | **标准体** |
| 一过性升高（恢复） | 曾≥140/90 | 已恢复正常≥2年 | 无 | 无 | **0** | **标准体** |
| 一过性升高（临界恢复） | 曾≥140/90 | 已接近正常 | 无 | 无 | 0~+25 | 标体或轻加费 |
| 一过性升高+肥胖 | 曾≥140/90 | 恢复或临界 | 无 | 无 | +25~+50 | 加费（代谢风险） |
| 间歇性升高 | 波动 | 持续存在 | 无 | 无 | +25~+50 | 加费 |
| 确诊高血压 | ≥140/90 | 持续 | 有/无 | 无 | 按分级 | 见高血压表 |

> **曹治家案例标注**: 2017年单次165/99 → 2021年130/81 → 当前130/90，无服药史，无靶器官损害 → **白大衣高血压/一过性升高**，当前舒张压90处于临界值。核保评点0（标准体），但因BMI 29.7叠加组合#15（高血压+肥胖），综合评点+25。

**三、核保关键区分**

> ⚠️ **"曾经有过血压升高" ≠ "高血压病史"**。核保必须区分：
> - **一过性升高**：单次或少数几次异常，后续自行恢复正常，无治疗 → 标准体
> - **白大衣高血压**：诊室升高但诊室外正常（需ABPM证实）→ 标准体
> - **确诊高血压**：持续性升高，需药物控制 → 加费+除外
>
> 保险健康告知中的"是否患有高血压"通常指**确诊且需要治疗**的高血压。一过性升高在如实告知后可争取标准体。

**血糖/糖尿病**

| 状态 | HbA1c | 评点 | 核保策略 |
|------|-------|------|---------|
| 正常（空腹<6.1mmol/L） | <5.7% | 0 | 标准体 |
| 糖尿病前期（6.1-6.9） | 5.7~6.4% | +25 | 轻度加费，嘱定期监测 |
| 2型糖尿病，控制良好（<7.0） | <7.0% | +75~+100 | 加费+糖尿病相关重疾除外 |
| 2型糖尿病，控制一般（7.0-8.0） | 7.0~8.0% | +100~+150 | 高额加费+除外 |
| 2型糖尿病，控制差（>8.0）或有并发症 | >8.0% | Dec | 拒保 |
| 1型糖尿病 | 任意 | Dec | 拒保（重疾险） |
| 糖尿病并发症（视网膜病变/肾病/神经病变） | — | Dec | 拒保 |

#### 室性早搏专项评估（v2.7新增）

> **沉淀来源**: 陶玉峰案例（27岁男性/24h Holter 3759次室早，负担3.7%/292阵三联律/17阵二联律/心脏超声正常/2024年ECG频发室早三联律→2025年ECG正常→2026年Holter证实持续存在）。

**室早负担分层核保矩阵**:

| 24h室早负担 | 三联律/二联律 | 器质性心脏病 | NRS评点 | 核保策略 |
|:----------:|:-----------:|:----------:|:-------:|---------|
| <1%（<1000次/24h） | 无 | 无 | 0~+25 | 标准体或轻加费 |
| 1-3% | 偶发（<10阵） | 无 | +25~+50 | 加费承保 |
| **3-5%** | **频发（>100阵三联律）** | **无** | **+50~+75** | **加费+心血管除外或高加费** |
| 5-10% | 有/无 | 无 | +75~+100 | 高额加费+心血管除外 |
| >10% | 任意 | 无 | +100~+200 | 高加费或拒保 |
| 任意 | 任意 | **有（心超异常/心功能↓）** | **Dec** | **拒保** |

**再保室早风险评估参考**:

| 再保公司 | 负担>1% | 负担>3% | 负担>5% | 关键考量 |
|---------|:------:|:------:|:------:|---------|
| Munich Re | +25~50 | +50~75 | +75~100 | 重视三联律/二联律频率 |
| Swiss Re | Table 1-2 | Table 2-3 | Table 3-4 | 重视心脏超声结果 |
| Gen Re | +25~50 | +50~100 | +100~150 | 2年内复查趋势 |

| SCOR | +25~50 | +50~75 | +100 | 亚洲市场中等尺度 |

**Holter报告中关键核保指标**:
- 总负担（%）= 室早总数 / 总心搏数 × 100%
- 三联律阵数（>50阵 = 显著）
- 二联律阵数（>10阵 = 显著）
- 成对室早（>0 = 风险升级）
- 室速发作（>0 = 拒保级风险）
- 停搏/长RR间期（>2秒 → 延期）
- ST段改变（缺血性 → 按CAD评估）
- 最慢心率（<40bpm → 考虑窦房结功能）

**⚠️ 假阴性ECG陷阱**: 常规10秒静息心电图可能漏诊频发室早！2024年ECG显示频发室早三联律→2025年ECG显示"正常"→2026年Holter证实3759次/24h。静息ECG仅10秒，采样窗口极短，室早呈间歇性发作时可完全不被捕获。**核保规则：如有心律失常史，必须提交24h动态心电图；不可仅凭"正常ECG"认定已恢复**。

**动态心电图核保决策规则**:
1. 标准12导联ECG（10秒）vs 动态心电图（23-24小时）→ 检出率差可达100倍
2. **必须提交Holter的情形**：既往ECG有频发早搏/三联律/二联律记录；有心悸胸闷主诉但常规ECG正常；体检触诊发现心律不齐；<50bpm心动过缓需排除病理性
3. Holter正常≠ECG正常：**应以Holter为准**，ECG"正常"不能推翻Holter异常

#### 青年男性（<40岁）心血管特殊规则（v2.8新增）

> 沉淀来源：陶玉峰案例（27岁男性室早）+ 临床经验（青年猝死风险）

**一、运动员心脏 vs 病理性肥厚**

| 情况 | 心脏超声特征 | 核保策略 |
|------|-----------------|---------|
| 运动员心脏（生理性肥厚） | LVH但舒张功能正常，无家族史 | 0~+25（需运动史确认）|
| 肥厚型心肌病（HOCM） | 室间隔不对称肥厚，SAM现象 | Dec（拒保） |
| 致心律失常性右室心肌病（ARVC） | 右室扩大，T波倒置（V1-V3） | Dec |

> **核保关键**：青年男性 + 心脏超声示 LVH → 必须排除 HOCM（可猝死）。要求提供 **心脏MRI + 基因检测（MYH7/ACTN2等）**。

**二、青年男性晕厥评估**

| 晕厥原因 | 危险分层 | 核保策略 |
|----------|---------|---------|
| 血管迷走性晕厥（Vasovagal） | 低 | 0~+25（通常标体）|
| 情境性晕厥（咳嗽/排尿性） | 低 | 0~+25 |
| 心源性晕厥（心律失常） | 高 | +100~Dec（拒保）|
| 不明原因晕厥（<40岁） | 中 | +50~+100，需Holter+心脏超声+运动试验 |

> **核保规则**：<40岁不明原因晕厥 → 必须提供 **Holter+心脏超声+运动试验+直立倾斜试验**，排除心源性后方可考虑承保。

**三、青年男性心血管风险评估要点**

| 风险因素 | <40岁特殊性 | 核保操作 |
|---------|-------------|---------|
| 室早负担 3-5% | 年龄越小风险越高（无心肌适应性） | 建议运动试验，阴性可标体 |
| 血脂正常高值 | 青年期起始 → 长期累积ASCVD风险 | 需评估家族史+LDL亚分型 |
| 吸烟 | 青年吸烟 → 10年后心血管事件风险显著升高 | 需戒烟≥1年，否则加费 |
| 猝死家族史 | 提示遗传性心律失常/心肌病 | 需基因检测+心脏MRI |

#### ST段改变核保专项评估（v2.9新增）

> **沉淀来源**: 2026-06-13案例（女性/ST段II/III/aVF下移≥0.05mV + LDL-C 3.25边缘升高 + CK 337显著升高 / CK-MB正常 / 血压122/74正常），结合Swiss Re、Munich Re、Gen Re心血管核保指引。

**一、ST段改变的核保分层**

ST段改变是心电图最常见的异常之一，但核保意义差异极大——从"完全良性"到"提示心肌缺血需拒保"均有可能：

| ST段改变特征 | 导联分布 | 伴随因素 | NRS评点 | 核保策略 |
|:-----------|---------|---------|:-------:|---------|
| 非特异性ST-T改变 | 任意 | 无心血管风险因子 | 0~+25 | 标体或轻加费，建议复查 |
| **ST段下移≥0.05mV** | **下壁（II/III/aVF）** | **LDL-C↑或CK↑** | **+50~+75** | **加费+心血管除外，建议心脏彩超+运动平板** |

#### CK升高+CK-MB正常鉴别核保（v2.9新增）

> **沉淀来源**: 2026-06-13案例（CK 337 U/L ≈ 2.4×ULN, CK-MB 13.0正常, LDH 160正常, AST 22正常, ALT 14正常）。

**一、CK升高的核保鉴别路径**

肌酸激酶(CK)升高是体检中的常见意外发现，核保核心任务：**判断是心肌源性还是骨骼肌源性**。

```
CK升高 → 鉴别：
├── CK-MB↑ (>25 U/L) → 🔴 心肌损伤可能 → 延期至心内科确诊
│   ├── + cTnI/cTnT↑ → 心肌梗死/心肌炎 → Dec
│   ├── + 心脏超声异常 → 器质性心脏病 → 按具体疾病评估
│   └── + 心脏超声正常 → 可能一过性，观察3个月后重评
│
├── CK-MB正常 + 无心脏症状 → 🟡 骨骼肌源性（可能）
│   ├── + 近期剧烈运动 → 休息2周复查，正常后标体
│   ├── + 药物（他汀/贝特类） → 停药后复查，正常后标体
│   ├── + 肌肉损伤/肌病 → 明确诊断后按肌病评估
│   ├── + 甲减 → TSH↑ → 按甲减评估
│   └── + 原因不明 → 建议复查+排查，可延期
│
└── CK↑ + LDH↑ + AST↑ → 多酶升高 → 需排除肌病/肝病/药物性
```

**二、CK升高核保评点矩阵**

| CK水平 | CK-MB | 心脏超声 | 查明原因 | NRS评点 | 核保策略 |
|-------|:-----:|:-------:|:-------:|:-------:|---------|
| 1-2×ULN | 正常 | — | 运动/药物史 | 0~+25 | 标体或轻加费 |
| **2-3×ULN** | **正常** | **正常** | **未查明** | **+25~+50** | **加费，建议延期排查** |
| 2-3×ULN | 升高 | 正常 | 未查明 | 延期 | 需排除心脏源 |
| >3×ULN | 正常 | 正常 | 未查明 | +50~+75 | 加费+心血管除外 |
| >3×ULN | 升高 | — | — | 延期→Dec | 高概率心肌损伤 |
| 任意升高 | — | 异常（EF↓/室壁异常） | — | Dec | 器质性心脏病证据 |

> **本案例标注**: CK 337 (2.4×ULN) + CK-MB 13.0正常 + LDH 160正常 + AST 22正常 → 倾向于骨骼肌源性。但原因未明确，建议复查CK+CK-MB。核保评点+25~+50，与ST段改变折叠后心血管集群+50~+75。

**三、CK-MB正常的核保积极意义**

CK-MB（心肌型肌酸激酶同工酶）正常具有关键鉴别价值：
- CK-MB正常 + CK升高 → **强烈提示骨骼肌源**（运动、肌病、药物等）
- CK-MB正常 ≠ 完全排除心梗（需结合cTnI/cTnT，CK-MB在CK极度升高时可有微量交叉反应）
- **核保结论**: CK-MB正常是**积极因素**，可在心血管综合评点中降低25%

**二、动脉粥样硬化与ST改变的病理关联**

```
LDL-C边缘升高 ─→ 动脉粥样硬化启动 ─→ 心肌缺血(ST段改变)
                                            │
CK升高(CK-MB正常) ─→ 骨骼肌源性(偏向) ←────┘
                    (不能完全排除亚临床心肌损伤)
```

> **核保逻辑**: ST段改变与LDL-C升高属于同一动脉粥样硬化病理链条 → 不完全叠加（取高评点×1.3）。CK升高若为骨骼肌源则与ST段改变无直接病理关联；若为隐匿性心肌损伤则风险显著升级。在CK原因未明时，按**最不利于被保人原则**（保守评估），将CK纳入心血管综合评分。

**三、本案例（2026-06-13）完整心血管评估路径**

```
ST段改变（下壁II/III/aVF≥0.05mV）→ 基础评点 +50~75
    ↓ 叠加 LDL-C 3.25（边缘升高）
    → 同一动脉粥样硬化路径，×1.3 = +65~98
    ↓ 叠加 CK 337（CK-MB正常，骨骼肌源性偏向）
    → 不同机制可能，CK-MB正常正向修正 −25%
    ↓ 最终心血管集群（综合折叠）
    → +50~+75
```

> **说明**: CK升高虽与LDL升高非同一病理机制，但CK-MB正常大幅降低心脏源性概率，故综合评点经折叠+修正后向下收敛至+50~+75。**但前提是：CK在复查中恢复正常**。如复查仍>300 U/L且无合理解释，评点应上调至+75~+100。

| TBIL+DBIL同步升高 | 升高 | 异常 | +50~+100 | 梗阻性黄疸，需肝胆专科评估 |

**Gilbert综合征鉴别要点**:
- 孤立性非结合胆红素（间接胆红素）升高
- ALT/AST、GGT、ALP均正常
- 肝胆超声正常
- 无溶血证据（Hb、网织红细胞正常）
- 无临床症状（无黄疸、无腹痛）
- 遗传性（UGT1A1基因变异），人群发生率~5%

> **核保结论**: Gilbert综合征为**良性遗传性高胆红素血症**，不影响肝功能和预期寿命 → **标准体承保**。但需排除真性肝病：（1）若ALT/AST同步升高 → 按肝炎评估；（2）若DBIL升高为主 → 按梗阻性黄疸排查；（3）若Hb下降+网织红细胞升高 → 按溶血性贫血评估。

#### 尿隐血核保评估（v3.0新增）

> **沉淀来源**: 曹治家案例（2021年尿干化学隐血弱阳性±/镜检红细胞未见/无蛋白尿无白细胞/后续未复查）。

**一、尿隐血的核保鉴别路径**

```
尿隐血阳性 → 鉴别：
├── 干化学弱阳性（±）+ 镜检红细胞未见
│   → 假阳性或微量血红蛋白
│   → 核保：**标准体**（临床意义极低）
│
├── 干化学阳性（+）+ 镜检红细胞少量（1-3/HPF）
│   → 可能为生理性或轻微泌尿系问题
│   → 核保：标准体或轻加费（+25）
│   → 建议：复查确认是否持续
│
├── 干化学阳性（++以上）+ 镜检红细胞增多（>5/HPF）
│   → 血尿，需鉴别来源
│   ├── 肾小球源性（畸形红细胞+蛋白尿）→ 按肾炎评估
│   ├── 非肾小球源性（均一型红细胞）→ 结石/感染/肿瘤排查
│   └── 核保：延期至明确病因
│
└── 持续血尿+蛋白尿+高血压
    → 肾炎综合征可能
    → 核保：Dec（拒保）或按慢性肾炎评估
```

**二、尿隐血核保评点矩阵**

| 干化学结果 | 镜检红细胞 | 蛋白尿 | 持续性 | NRS评点 | 核保策略 |
|-----------|:---------:|:-----:|:-----:|:-------:|---------|
| **弱阳性（±）** | **未见** | **阴性** | **单次** | **0** | **标准体** |
| 弱阳性（±） | 未见 | 阴性 | 持续（多次） | 0~+25 | 标体，建议复查 |
| 阳性（+） | 未见 | 阴性 | 单次 | 0~+25 | 标体，建议复查 |
| 阳性（+） | 少量（1-3） | 阴性 | 单次 | +25 | 轻加费，建议随访 |
| 阳性（++） | 增多（>5） | 任意 | 任意 | 延期 | 需明确血尿原因 |
| 任意阳性 | 任意 | **阳性** | 任意 | +50~延期 | 肾脏损伤评估 |
| 任意阳性 | 任意 | 阳性+高血压 | 任意 | Dec | 肾炎综合征 |

> **曹治家案例标注**: 尿干化学弱阳性（±）+ 镜检红细胞未见 + 尿蛋白质阴性 + 单次 → **标准体**。干化学法的隐血检测基于过氧化物酶反应，灵敏度高，可因肌红蛋白、氧化剂污染等产生假阳性。镜检红细胞未见是最有力的阴性证据。

**三、核保关键经验**

> ⚠️ **尿干化学隐血 ≠ 血尿**。核保中应以**镜检红细胞**为准：
> - 干化学弱阳性+镜检阴性 = **假阳性，核保无影响**
> - 干化学阳性+镜检阳性 = **真性血尿，需进一步鉴别**
> - 核保关注的不是干化学结果的"阳性"或"阴性"，而是**是否有实际的肾脏/泌尿系统损伤证据**（蛋白尿、肾功能异常、畸形红细胞等）。

### 甲状腺癌术后核保专项评估

> **沉淀来源**: 基于戴薇案例（28岁女性/T1bN1bM0 PTC经典型/BRAF阴性/术后2年）的系统分析，结合Swiss Re、Munich Re、Gen Re、Hannover Re等再保手册及中国临床实践，提炼为可推广的核保经验规则。

#### 一、预后分层体系

甲状腺乳头状癌核保评估需综合三个维度：

**维度1：AJCC 8th TNM分期（注意年龄交互：<55岁最高仅Stage II）**

| 风险组 | TNM组合 | 5年DSS | 核保含义 |
|--------|---------|--------|---------|
| 极低危 | T1aN0M0（微小癌≤1cm） | >99% | 局部切除后可考虑标体 |
| 低危 | T1b-T2N0M0 | >98% | 术后3年可标体或仅除外 |
| 中危 | T1-3N1aM0（中央区LN+） | ~95% | 术后5年除外+加费 |
| **中高危** | **T1-3N1bM0（侧颈LN+）** | **~90%** | **术后5年除外+加费50-100%** |
| 高危 | T4或M1 | <80% | 延期/拒保 |

**维度2：ATA复发风险分层**

| 分层 | 标准 | 5年复发率 | 核保处理 |
|------|------|:--------:|---------|
| 低危 | N0, 无血管侵犯, 切缘阴性, 无ETE | 1-5% | 术后3年可标体 |
| 中危 | N1(≤5个微小LN), 微小ETE, 血管侵犯, BRAF+ | 5-20% | 术后5年除外+加费 |
| **高危** | **N1(>5个LN或>3cm), 大范围ETE, 远处转移** | **>20%** | **延期/高额加费** |

**维度3：戴薇案例标注** — T1bN1b（侧颈LN 3/33阳性）→ 中高危组，5年复发率约15-20%

#### 二、术后时间轴核保决策矩阵

| 术后年限 | T1aN0 极低危 | T1bN0 低危 | T1N1a 中危 | **T1bN1b 中高危** | T3-4/高危 |
|---------|:-----------:|:---------:|:---------:|:----------------:|:--------:|
| <1年 | 延期 | 延期 | 延期 | **延期/拒保** | 拒保 |
| 1-2年 | 除外+0~25% | 延期/除外+25% | 延期 | **延期** | 拒保 |
| 2-3年 | 除外或标体 | 除外+25% | 除外+50-75% | **除外+50-100%** | 延期 |
| 3-5年 | 标体 | 除外或标体 | 除外+25-50% | **除外+50-75%** | 除外+100% |
| 5-10年 | 标体 | 标体 | 标体或除外 | **除外+25-50%** | 除外+50-75% |
| >10年 | 标体 | 标体 | 标体 | **除外或标体** | 除外+25% |

> **险种差异**: 以上为重疾险标准。医疗险：永久除外甲癌+相关并发症（多数公司），寿险：综合评估LN+对远期生存的影响，尺度通常更保守。

#### 三、分子病理对核保的调整

| 分子标志物 | 状态 | 对核保的影响 |
|-----------|------|------------|
| **BRAF V600E** | 阳性(+) | 复发风险×1.5-2.0 → 加费上调25%或延长观察期1-2年 |
| **BRAF V600E** | 阴性(-) | 有利因素（约40-60%PTC为阳性），但**不改变N1b的中高危属性** |
| **TERT启动子** | 突变(+) | 高风险标志 → 建议拒保或极其严格核保 |
| **TERT启动子** | 野生型 | 标准评估 |

#### 四、术后监测指标的核保意义

| 指标 | 理想状态 | 警戒值 | 核保判断 |
|------|---------|--------|---------|
| **Tg**（甲状腺球蛋白） | <0.2 ng/mL（测不出） | >2 ng/mL或上升趋势 | 测不出=无残留/复发→**有利**；上升→**延期** |
| **TgAb** | 阴性(<4 IU/mL) | 阳性或升高 | 阳性干扰Tg测定，需影像证实无复发 |
| **TSH抑制** | 高危组<0.1-0.5 mU/L | >0.5（抑制不足） | 达标=良好依从性；未达标=复发风险增加，建议优化后再投保 |
| **颈部超声** | 无异常LN | 可疑LN(>1cm, 钙化, 无淋巴门) | 反应性LN(<1cm,有淋巴门)→良性，无需额外除外 |
| **I-131 WBS** | 阴性 | 阳性浓聚 | 如已行RAI，阴性为有利证据 |

> 戴薇案例实践：Tg 3.55未测不出 + TSH 1.905抑制不足 → 建议优化优甲乐剂量，TSH降至<0.5后提供最新甲功数据再评估更有利。

#### 五、甲状腺癌+肺结节共病专项评估

先排除肺转移，再行核保：

| 肺结节特征 | 甲癌风险分层 | 核保处理 |
|-----------|:-----------:|---------|
| <6mm，稳定≥2年 | 任何 | 按甲癌单独评估，肺结节标体（稳定2年排除转移） |
| 6-10mm，稳定≥2年 | 低危 | 按甲癌评估+呼吸系统除外 |
| 6-10mm，稳定≥2年 | **中高危(N1b)** | **除外甲癌+呼吸系统疾病，加费+25%** |
| 任何大小，进展中 | 任何 | 延期至排除肺转移 |
| >10mm或恶性特征 | 任何 | 延期，建议穿刺活检 |

**转移鉴别**：甲癌肺转移→双肺多发粟粒结节+I-131浓聚；良性微小结节→单发、边界清、长期稳定、I-131不浓聚。

#### 六、智能核保 vs 人工核保策略差异

| 渠道 | 适用条件 | 策略建议 |
|------|---------|---------|
| **智能核保**（复兴联合/好医保等） | T1N0术后3年+最适用 | **首选！** 可能仅除外不加费（智能规则通常不区分N分期） |
| **智能核保** | N1b术后2-5年 | 可尝试，但可能直接拒保 |
| **人工预核保**（经纪人） | 所有分层 | 附完整病理+随访资料，强调有利因素 |
| **线下人工核保**（传统保司） | 中高危 | 术后满3-5年再提交更稳妥 |
| **惠民保/防癌险** | 被拒保后 | 不限健康状况，兜底保障 |

> **关键经验**：互联网智能核保对PTC术后可能比传统人工核保**更宽松**，因其规则通常仅区分"是否癌症史+术后年限"，不细分N0/N1a/N1b → 术后2年N1b可能仅被"除外甲状腺癌"不加费。但各产品规则可能迭代，不可依赖。

### 十一、甲状腺结节射频消融术后核保专项评估

> **沉淀来源**: 基于王烨佳案例（42岁女性/甲状腺双叶结节射频消融术后/TI-RADS 3类/TPOAb 66.50 IU/mL）的系统分析，结合慕再MIRA核保手册、Swiss Re、Gen Re再保指引。

#### 一、射频消融 vs 手术切除的核保差异

| 维度 | 射频消融（RFA） | 手术切除（甲状腺全切/次全切） |
|------|:-----------:|:-----------------------:|
| 创伤程度 | 微创，保留腺体 | 较大，部分/全部切除 |

| 复发风险 | 较高（残留组织可再生） | 较低（无/少残留组织） |

| 核保评点 | +25~+75（视残留+分级） | 按TNM分期标准流程 |

> **关键经验**: 射频消融术后核保比传统手术切除**更保守**，因为：（1）缺乏完整病理报告；（2）残留腺体组织有再生/新发结节风险；（3）消融效果评估依赖影像而非病理。核保时需额外关注残留结节大小和TI-RADS分级。

### 十三、血液系统特殊指标核保意义

> **沉淀来源**: 基于王烨佳案例（嗜酸性粒细胞突升至18.5%，前两年3.2-3.8%）。

#### 一、嗜酸性粒细胞偏高核保意义

| 情况 | 绝对值(×10⁹/L) | 百分比 | 核保影响 |
|------|:-------------:|:-----:|---------|

> **王烨佳案例标注** — 嗜酸性粒细胞1.25×10⁹/L（参考<0.52，约2.4倍升高），百分比18.5%（前两年3.2-3.8%突然升高）→ 需排查原因（过敏原检测、寄生虫筛查、自身免疫指标），排查结果正常后可标体。**不排查前不建议直接投保**，因嗜酸性粒细胞增多可能提示潜在疾病。

**嗜酸性粒细胞升高的鉴别排查路径:**
```
嗜酸性粒细胞升高 → 排查：
├── 过敏性疾病（哮喘、过敏性鼻炎、药物过敏、食物过敏）
├── 寄生虫感染（在中国相对少见，但需排除）
├── 自身免疫性疾病（桥本？类风湿？炎症性肠病？）
├── 皮肤疾病（湿疹、荨麻疹）
├── 药物反应（近期新用药？）
├── 肿瘤（需排除，尤其合并体重下降时）
└── 特发性嗜酸性粒细胞增多症（排除性诊断）
```

## 综合风险评分机制

当存在多种异常情况时，使用累加评分（综合评点 = Σ各项评点），但须注意：

1. **相关性折叠原则**：同一病理机制（如代谢综合征：肥胖+高血压+糖尿病）不简单叠加，最高项评点加权系数 × 0.5~0.7。
2. **重疾险除外责任上限**：通常建议不超过 3 项除外责任，超过则考虑提高加费或拒保。
3. **综合评点上限**：综合评点超过 400% 则自动进入经验核保（个案评估）。
4. **东亚心血管综合风险**：参考 Framingham 亚裔修正版评分，10年心血管事件风险 <10% 通常可标体，≥20% 通常考虑加费。

## 共病组合核保评估（34组）（v2.9扩充）

> 共病组合（Comorbidity）指同一被保人同时存在≥2种健康异常。核保需评估组合风险是否高于单一疾病简单相加。

### 折叠规则（通用）

- **同一病理机制**（如代谢综合征：肥胖+高血压+糖尿病）→ 不完全叠加，取最高评点×0.6~0.8
- **不同系统**（如甲状腺+肺结节）→ 独立评估，简单相加
- **自身免疫叠加**（如桥本+其他自身免疫）→ 需额外评估活跃程度，评点×1.2~1.5
- **内分泌轴关联**（如甲状腺+乳腺）→ 同一HPA轴，取较高评点×1.2

| 36 | 一过性高血压(恢复)+BMI 29.7 | 代谢风险(组合#15变体) | 取高（BMI+25） | 标体或轻加费（血压恢复降低风险） |

> **折叠规则实操**：表中"取高+25%"意为两病评点中取较高者，再追加25%额外评点（关联风险）。"相加×1.2"意为两病评点相加后×1.2（协同风险）。

### 十五、多系统微小结节共病专项评估（v2.9新增）

> **沉淀来源**: 2026-06-13案例（甲状腺TI-RADS 2类 2×1mm + 双侧乳腺BI-RADS 3类多发 5×3mm+4×2mm + 右乳钙化灶 + 右肺微小结节2mm + 肺尖陈旧性病灶/胸膜粘连），覆盖甲状腺/乳腺/肺三个系统良性微小结节的共病评估。

**一、"微小结节三联征"的核保含义**

当同一被保人体检报告中出现≥2个系统的微小结节时，核保需区分：

| 结节特征 | 核保含义 | 处理策略 |
|---------|---------|---------|
| 各系统结节均为**良性分级**（TI-RADS≤3/BI-RADS≤3/Lung-RADS≤2） | 体质性结节倾向（体质性多结节综合征倾向），非恶性 | 独立评估，不累加 |
| 各系统结节**大小均极小**（甲状腺<5mm/乳腺<10mm/肺<6mm） | 微小结节，恶性概率极低 | 取高评点，不额外加费 |
| 仅部分系统为高危结节 | 聚焦高危结节，良性结节不升级 | 按高危系统评点 |
| 同时存在**钙化灶**（多系统） | 炎症/退行性愈后改变，非恶性 | 标体 |

> **本案例（2026-06-13）**: 甲状腺TI-RADS 2(2mm) + 乳腺BI-RADS 3(5mm/4mm) + 肺微小结节(2mm) + 右乳钙化灶 → 三个系统均为**良性+微小**特征，属于典型的"体质性多系统微小结节"模式。独立评估即可，不叠加加费。

**二、多系统结节核保矩阵**

| 系统组合 | 各分级组合 | 折叠规则 | 除外策略 | 加费 |
|---------|:---------|:-------:|---------|:---:|
| 甲状腺+乳腺 | TI-RADS 2 + BI-RADS 3 | 取高（乳腺+25） | 除外乳腺 | 无 |
| 甲状腺+肺 | TI-RADS 2 + Lung-RADS 2 | 取高（肺+25） | 除外肺 | 无 |
| 乳腺+肺 | BI-RADS 3 + Lung-RADS 2 | 取高+25% | 除外乳腺+肺 | 无 |
| **甲状腺+乳腺+肺** | **TI-RADS 2+BI-RADS 3+Lung-RADS 2** | **取高+25%（仅乳腺与肺有协同）** | **除外乳腺+肺（甲状腺标体）** | **无额外加费** |
| 甲状腺+乳腺+肺 | TI-RADS 3+BI-RADS 3+Lung-RADS 2 | 取高×1.5 | 除外三系统 | +25~+50 |
| 任一系统为≥4级 | +任意 | 按高危系统评估 | 延期或拒保 | N/A |

**三、结节共存时的干预阈值调整**

当多系统结节共存时，核保关注的重点不是结节数量，而是**每个系统独立的风险等级**：

```
多系统结节共存 → 核保决策逻辑:
├── 所有系统均为良性分级 → 独立评估，取最高评点
│   └── 乳腺BI-RADS 3评点(+25) > 甲状腺TI-RADS 2(0) + 肺Lung-RADS 2(0)
│       → 最终评点 = +25（仅乳腺）
├── 任一系统为交界性 → 聚焦交界性系统，良性系统不升级
│   └── 如BI-RADS 4 + TI-RADS 2 → 按BI-RADS 4评估(延期/活检)
└── 多系统均为交界性 → 叠加评估，考虑遗传性肿瘤综合征可能
    └── 如BI-RADS 4 + TI-RADS 4 → 延期，建议基因检测(BRCA/PTEN等)
```

**四、钙化灶在多系统结节中的核保意义**

| 钙化灶类型 | 位置 | 核保含义 | 处理 |
|-----------|------|---------|------|
| 乳腺点状钙化（BI-RADS 3类结节同存） | 乳腺 | 可能为退化性改变/分泌性钙化 | 与BI-RADS 3结节一并评估 |
| 肺尖陈旧性钙化灶 | 肺尖 | 陈旧性结核/炎症愈后 | 标体（需确认无活动性结核） |
| 胸膜粘连+钙化 | 胸膜 | 陈旧性胸膜炎（结核性可能） | 确认无活动性后标体 |
| 多系统同时存在钙化灶 | 乳腺+肺 | **全身性营养不良性钙化倾向？** | 独立评估，不叠加 |

> **本案例（2026-06-13）**: 右乳钙化灶+两肺尖钙化灶+胸膜粘连 → 虽为多系统，但均为陈旧性/退行性改变，无活动性感染证据。核保影响：乳腺钙化与BI-RADS 3结节合并评估（不独立加费），肺部钙化按陈旧性病灶处理（标体）。

**五、再保对多系统结节的态度**

| 再保公司 | 多系统微小结节立场 |
|---------|-----------------|
| **Munich Re** | 甲状腺+乳腺+肺三系统结节→建议独立评估各系统，不因"多系统"升级风险 |
| **Swiss Re** | 多系统微小结节→关注各系统分级，良性结节不累加 |
| **Gen Re** | 强调"体质性多结节"不等于"癌症综合征"，需区分分级 |
| **SCOR** | 亚洲女性多系统良性结节常见，无需过度评估 |

## 延期标准

以下情况需延期投保（PP）：
- 近3个月内曾住院或手术（非紧急情况）
- 确诊后尚在检查随访阶段，病情性质未明确（如肺结节、甲状腺结节高危）
- 正在接受放疗、化疗或免疫治疗
- 妊娠期（通常延期至产后6周以上）
- 急性病或慢性病急性发作期（如急性乙肝、急性心梗）
- 某些慢性病确诊 <1年，病情稳定性未知

## 处理医学资料的指引

**1. 血液检验报告**
- 关注：血常规（Hb、WBC、PLT）、肝功能（ALT/AST/GGT/白蛋白/胆红素）、肾功能（肌酐/eGFR/尿酸）、血脂（TC/TG/LDL/HDL）、血糖/HbA1c、肿瘤标志物（AFP/CEA/CA125/CA199/PSA等）、乙肝五项、甲功五项
- 若报告显示异常但无历史对比，说明需要追踪随访记录

**2. 影像报告**
- 超声：关注器官大小、回声性质、占位病变性质（良恶性描述）、血流信号
- CT/MRI：关注结节大小（mm）、密度（磨玻璃/实性）、边界、增强特征、淋巴结情况
- 若报告仅有文字描述无图像，仍可依据描述进行评估

**3. 病理报告**
- 这是最权威的诊断依据
- 关注：组织学类型、分级（Grade）、浸润深度、切缘、脉管/神经浸润、免疫组化

**4. 体检报告**
- 按系统逐项提取异常，关注"建议进一步检查"项目
- **PDF提取方法**（按优先级）：
  1. 文字型PDF → PyMuPDF (fitz) 直接提取（首选，毫秒级）
  2. 扫描件PDF → PaddleOCR → pdf-ocr-layout → Tesseract
- **异常标记自动识别**：↑↓△H L 阳性 偏高 偏低 BI-RADS/TI-RADS分级≥3
- **终检结论**是核保评估最核心的依据，务必完整提取
- **多次报告对比**：当存在历史体检报告时，必须对比关键指标趋势（恶化/好转/新发/消失）

**5. 门诊/住院病历**
- 关注确诊日期、治疗经过、手术记录、出院诊断、随访状态

### PaddleOCR v3+ 医学文档批量OCR实战（v2.9新增，基于2026-06-13案例）

> **沉淀来源**: 19页体检报告PNG扫描件→PaddleOCR v3+ (PP-OCRv5_server) 批量OCR完整工作流。API适配经历3次迭代试错后方可运行。

**一、PaddleOCR v3+ API迁移要点**

PaddleOCR从v2升级到v3+后，API发生断裂式变化，旧代码**直接报错**不可用：

| 方面 | v2 (旧) | v3+ (新) | 注意 |
|------|---------|----------|------|
| 核心方法 | `ocr.ocr(img_path)` | `ocr.predict(img_path)` | 方法名变化，旧名报错 |
| 返回格式 | `list[list[bbox, (text, conf)]]` | `list[OCRResult]` 对象，含`rec_texts`/`rec_scores`/`dt_polys`字段 | 结构完全不同 |
| `use_angle_cls` | ✅ 支持 | ❌ 已移除（自动内置） | 传入会报unknown parameter |
| `show_log` | ✅ 支持 | ❌ 已移除 | 传入会报unknown parameter |
| `cls` | ✅ 支持 | ❌ 已移除 | — |
| 最小初始化 | `PaddleOCR(use_angle_cls=True, lang='ch')` | `PaddleOCR(lang='ch')` | 极简参数即可 |
| 模型缓存 | `~/.paddleocr/` | `~/.paddlex/official_models/` | 路径变化 |

**二、批量OCR Python脚本核心代码**

```python
from paddleocr import PaddleOCR

# v3+ 最小初始化（不要传入use_angle_cls/show_log等已废弃参数）
ocr = PaddleOCR(lang='ch')

def extract_text_from_result(result):
    """从PaddleOCR v3+ predict()结果提取文字"""
    lines = []
    confidences = []
    
    # result is list[OCRResult], each OCRResult has:
    #   rec_texts: list[str] - recognized text lines
    #   rec_scores: list[float] - confidence scores
    #   dt_polys: list[np.ndarray] - detection polygons
    for item in result:
        if hasattr(item, 'rec_texts'):
            for i, txt in enumerate(item.rec_texts):
                conf = item.rec_scores[i] if i < len(item.rec_scores) else 0
                lines.append(txt)
                confidences.append(conf)
    
    text = '\n'.join(lines)
    avg_conf = sum(confidences) / len(confidences) if confidences else 0
    return text, avg_conf, len(lines)

# 批量处理
for page_num, filepath in enumerate(pages, 1):
    result = ocr.predict(filepath)  # NOT ocr.ocr()!
    text, confidence, line_count = extract_text_from_result(result)
```

**二、OCR.space API核心代码**

```python
import requests
import base64

API_URL = "https://api.ocr.space/parse/image"
API_KEY = "helloworld"  # 免费测试key

with open(image_path, "rb") as f:
    img_data = base64.b64encode(f.read()).decode()

payload = {
    "base64Image": f"data:image/jpeg;base64,{img_data}",
    "language": "chs",       # 中文简体
    "isOverlayRequired": "false",
    "OCREngine": "2",        # Engine 2效果更好
}

headers = {
    "apikey": API_KEY,
    "Content-Type": "application/x-www-form-urlencoded",
}

resp = requests.post(API_URL, data=payload, headers=headers, timeout=30)
result = resp.json()

for pr in result.get("ParsedResults", []):
    text = pr.get("ParsedText", "")
    print(text)
```

```python
        corrected_name = info.filename.encode('cp437').decode('gbk')
        # 提取到正确路径
        data = zf.read(info.filename)
        with open(os.path.join(output_dir, corrected_name), 'wb') as f:
            f.write(data)
```

> **关键经验**: Windows环境下，中文ZIP文件（尤其是国产压缩软件如好压/WinRAR制作的ZIP）常使用GBK编码。Python zipfile模块默认用CP437解码，导致中文文件名显示为乱码（如`▓▄╓╬╝`）。使用`encode('cp437').decode('gbk')`可正确还原中文文件名。

**核保材料处理标准流程：**
1. 解压ZIP（修正中文编码）
2. 提取docx/txt文字材料（markitdown或PyMuPDF）
3. 对图片材料执行OCR（PaddleOCR → OCR.space → Tesseract降级）
4. 合并文字+OCR结果，构建完整医学信息摘要
5. 按21病种归类 → 核保规则匹配 → 生成评估报告

**五、批量OCR踩坑清单**

| 坑 | 现象 | 解决方案 |
|----|------|---------|
| 使用`ocr.ocr()` | `AttributeError: 'PaddleOCR' object has no attribute 'ocr'` | 改用`ocr.predict()` |
| 传入`use_angle_cls=True` | `TypeError: unexpected keyword argument` | 删除该参数（v3+已内置） |
| 传入`show_log=False` | 同上 | 删除 |
| 返回格式误解 | 按v2的`list[list[bbox, (text, conf)]]`解析→空结果 | 按`rec_texts`/`rec_scores`字段解析 |
| 首页测试置信度0% | 字段解析错误导致conf全部为0 | 测试第一页确认返回结构后再批量 |
| 页面处理顺序 | 文件系统排序≠页码顺序 | `re.match(r'page_(\d+)', f)`显式排序 |

### 体检报告结构认知

中国主流体检报告通常包含以下模块（顺序可能不同）：

```
1. 封面/头部：姓名、性别、年龄、体检日期、体检编号
2. 健康问卷：既往史、手术史、外伤史、过敏史
3. 一般检查：身高、体重、BMI、血压
4. 内科/外科/妇科/眼科/耳鼻喉/口腔
5. 检验报告：尿常规、血常规、生化（肝功/肾功/血糖/血脂）、免疫发光（甲功/肿瘤标志物）
6. 影像报告：超声（甲状腺/乳腺/腹部/妇科）、CT、X线
7. 心电图
8. 终检结论（最重要！体检机构综合总结）
9. 影像附件页（10-20页，应跳过不提取）
```

### 多次报告对比核保评估要点

当存在多次体检报告时，核保评估必须关注趋势变化：

| 变化类型 | 核保影响 | 示例 |
|---------|---------|------|
| 增生→结节 | 风险升级，需重新评估 | 乳腺小叶增生→BI-RADS 3类（张瑜）；乳腺腺病→新增形态不规则结节（王烨佳） |
| 正常→超标 | 新增加费/除外因素 | TC 5.45→6.30, LDL 3.36→4.21（张瑜）；TBIL 17.6→23.6↑（陶玉峰） |
| 阳性→未复查 | 需补充最新结果 | HPV16/18+2025检出，2026未查（张瑜） |
| 未告知→新告知 | 需专项评估 | 先心术后2026年新告知（张瑜） |
| 结节稳定 | 可排除延期 | 肺结节需CT随访确认稳定；肺结节4mm稳定2年→标体（陶玉峰） |
| 指标好转 | 可缩小除外/降低加费 | TSH 4.15→2.631正常化（张瑜）；BP 130/62→119/68（陶玉峰） |
| 新发斑块 | ASCVD风险启动 | 颈动脉正常→IMT增厚→新发斑块（王烨佳，42岁） |
| 指标突升 | 需排查原因后再评估 | 嗜酸性粒细胞3.2%→18.5%（王烨佳） |
| 新发现抗体异常 | 自身免疫评估 | TPOAb首次检出66.50（7×参考值）→桥本疑（王烨佳） |
| **ECG正常→Holter异常** | **⚠️ 假阴性ECG陷阱** | **2024 ECG频发室早三联律→2025 ECG正常→2026 Holter 3759次/24h（陶玉峰）** |
| **无→新发器官异常** | **新增加费/除外因素** | **双肾正常→左肾结石5mm（陶玉峰）；前列腺正常→钙化灶（陶玉峰）** |
| **阳性→转阴（已治愈）** | **可移除除外因素** | **HP阳性→阴性（陶玉峰）；中性粒细胞1.72↓→1.97正常（陶玉峰）** |

#### ⚠️ 假阴性ECG陷阱深度分析

> **沉淀来源**: 陶玉峰案例——2024年瑞慈ECG显示"频发室性早搏呈三联律"→2025年同济ECG显示"窦性心动过缓"→2025年瑞慈ECG显示"正常心电图"→2026年Holter证实3759次室早/24h（3.7%负担）、292阵三联律、17阵二联律。**室早从未消失，只是短程ECG未捕获**。

**核心教训**:
1. 标准12导联ECG仅采样10秒，室早呈间歇性发作时可完全不被捕获
2. 如果被保人仅提交2025年"正常ECG"而不提供2024年异常ECG和2026年Holter → 核保将错误地按"心律失常已自愈"评定为标准体
3. **核保必须核实完整医疗记录链**——单时间点"正常"不代表疾病消失
4. 智能核保产品（仅依赖被保人主动告知）存在漏检风险——ECG看似正常但Holter确有异常

**核保规则**: 
- 有既往心律失常史 → **必须提交24h动态心电图**（不可仅凭静息ECG）
- ECG"正常"不能推翻Holter异常
- 如无法提供Holter → 按最不利于被保人的历史记录评估（即按异常ECG评定）
