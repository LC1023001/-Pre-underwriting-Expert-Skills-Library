# 重大疾病险智能核保系统

## 系统定位

本 Skill 依据全球前十大再保险公司核保手册（慕尼黑再保险 MIRA/NRS 系统、瑞士再保险、汉诺威再保险、中再寿险等）与中国大陆主要人身险公司核保规范（平安、太平洋、中国人寿、泰康、友邦等），结合《重大疾病保险的疾病定义使用规范（2020年修订版）》，专为东亚人群（中国大陆、港澳台、日韩）健康风险特征优化，提供科学、合理、可落地的重大疾病险核保评估。

> 重要声明：本系统输出为辅助评估意见，最终核保决定须由持牌核保人员依据保险公司具体核保规程作出。

## 核保工作流程

当用户提供医学资料请求核保评估时，按以下步骤执行：

### 步骤一：信息收集与整理

收集并整理以下维度的信息：

**基本信息**
- 年龄、性别（不同年龄段风险权重不同）
- 拟投保险种（重疾险保额、保险期间）
- 家族病史（尤其一级亲属恶性肿瘤、心血管、糖尿病史）

**医学资料来源**
- 体检报告（年度健康体检）
- 医院病历（门诊/住院记录）
- 检验报告（血常规、生化、肿瘤标志物等）
- 影像报告（超声、CT、MRI、X线）
- 病理报告（活检结果）
- 手术记录
- 出院小结

### 步骤二：NRS 数字评点评估

采用改良 NRS（Numerical Rating System）数字评点法，参考慕再核保手册：

**评点基准**：以标准体死亡率/发病率为 100%，额外风险用百分点表示。

**重大疾病险评点决策标准**：
- **0点（标准体）** → 标准保费承保（标体）
- **+25 ~ +75点** → 轻度加费承保（额外保费约 +25%~+75%）
- **+100点** → 中度加费承保（额外保费 +100%，即双倍保费）
- **+100点 + 除外责任** → 组合条件承保
- **+200点以上** → 高度加费或拒保（重疾险通常不超过3项除外）
- **延期（PP）** → 病情不稳定，需观察后重新评估
- **拒保（Dec）** → 风险过高，无法承保

> 重疾险核保原则：重大疾病险的评点通常为 0（标准体）或 +100%（双倍费率），高风险情况优先采用"除外责任"而非大额加费，且建议除外责任不超过3项。

### 步骤三：东亚人群风险修正

在 NRS 基础上，叠加以下东亚人群特异性风险调整：

**高发疾病上调权重**（与欧美标准相比）：
- 乙型肝炎（HBV）携带/慢性乙肝：风险权重 ×1.5
- 肝细胞癌（与 HBV 高度相关）：风险权重 ×1.8
- 鼻咽癌（东南亚及华南人群高发）：风险权重 ×2.0
- 胃癌（与 Hp 感染、饮食习惯相关）：风险权重 ×1.5
- 食管癌（华北/中原地区高发）：风险权重 ×1.4
- 2型糖尿病（亚裔 BMI 阈值更低）：BMI≥23 即启动代谢风险评估
- 代谢综合征（脂肪肝、高脂血症）：综合评分
- 地中海贫血（南方省份）：需关注基因携带状态

**风险下调（相对欧美人群）**：
- 前列腺癌：东亚发病率低，风险权重 ×0.5
- 黑色素瘤：东亚发病率低，评估可适当宽松
- 心血管事件（冠心病）：50岁以下东亚男性相对风险低于欧美，但不得忽视高血压和脑卒中风险

### 步骤四：核保结论输出

生成格式化核保意见书（见"输出模板"章节）。

## 核保评估规则库

### 一、基础生命指标评估

**BMI（东亚适用标准）**

| BMI 范围 | 评点 | 说明 |
|---------|------|------|
| 18.5 ~ 22.9 | 0（标准） | 东亚理想体重 |
| 23.0 ~ 24.9 | 0（标准，观察） | 超重预警，关注血压血糖 |
| 25.0 ~ 27.4 | +25 | 轻度超重，关注代谢指标 |
| 27.5 ~ 29.9 | +50 | 中度超重，需关注合并症 |
| 30.0 ~ 34.9 | +100 | 肥胖，建议加费或组合除外 |
| ≥35.0 | +150 ~ Dec | 重度肥胖，高风险 |
| <18.5（体重过低） | +25 | 需排除慢性消耗性疾病 |

> 东亚修正：中国居民 BMI≥24 即定义为超重，BMI≥28 为肥胖，比欧美标准低。

**血压（中国标准）**

| 血压状态 | 评点 | 核保策略 |
|---------|------|---------|
| <120/80 mmHg | 0 | 标准体 |
| 120-139/80-89（高正常） | 0 ~ +25 | 观察，关注是否升级 |
| 1级高血压（140-159/90-99），药物控制达标 | +25 ~ +75 | 加费，除外高血压相关重疾 |
| 1级高血压，未用药或控制不佳 | +75 ~ +100 | 加费+除外 |
| 2级高血压（160-179/100-109） | +100 ~ Dec | 加费+除外或延期 |
| 3级高血压（≥180/110） | Dec | 拒保 |
| 高血压合并靶器官损害（心/脑/肾） | Dec | 拒保 |

**血糖/糖尿病**

| 状态 | HbA1c | 评点 | 核保策略 |
|------|-------|------|---------|
| 正常（空腹<6.1mmol/L） | <5.7% | 0 | 标准体 |

## 综合风险评分机制

当存在多种异常情况时，使用累加评分（综合评点 = Σ各项评点），但须注意：

1. **相关性折叠原则**：同一病理机制（如代谢综合征：肥胖+高血压+糖尿病）不简单叠加，最高项评点加权系数 × 0.5~0.7。
2. **重疾险除外责任上限**：通常建议不超过 3 项除外责任，超过则考虑提高加费或拒保。
3. **综合评点上限**：综合评点超过 400% 则自动进入经验核保（个案评估）。
4. **东亚心血管综合风险**：参考 Framingham 亚裔修正版评分，10年心血管事件风险 <10% 通常可标体，≥20% 通常考虑加费。

---

## 延期标准

以下情况需延期投保（PP）：
- 近3个月内曾住院或手术（非紧急情况）
- 确诊后尚在检查随访阶段，病情性质未明确（如肺结节、甲状腺结节高危）
- 正在接受放疗、化疗或免疫治疗
- 妊娠期（通常延期至产后6周以上）
- 急性病或慢性病急性发作期（如急性乙肝、急性心梗）
- 某些慢性病确诊 <1年，病情稳定性未知

## 处理医学资料的指引

**1. 血液检验报告**
- 关注：血常规（Hb、WBC、PLT）、肝功能（ALT/AST/GGT/白蛋白/胆红素）、肾功能（肌酐/eGFR/尿酸）、血脂（TC/TG/LDL/HDL）、血糖/HbA1c、肿瘤标志物（AFP/CEA/CA125/CA199/PSA等）、乙肝五项、甲功五项
- 若报告显示异常但无历史对比，说明需要追踪随访记录

**2. 影像报告**
- 超声：关注器官大小、回声性质、占位病变性质（良恶性描述）、血流信号
- CT/MRI：关注结节大小（mm）、密度（磨玻璃/实性）、边界、增强特征、淋巴结情况
- 若报告仅有文字描述无图像，仍可依据描述进行评估

**3. 病理报告**
- 这是最权威的诊断依据
- 关注：组织学类型、分级（Grade）、浸润深度、切缘、脉管/神经浸润、免疫组化

**4. 体检报告**
- 按系统逐项提取异常，关注"建议进一步检查"项目
- **PDF提取方法**（按优先级）：
  1. 文字型PDF → PyMuPDF (fitz) 直接提取（首选，毫秒级）
  2. 扫描件PDF → PaddleOCR → pdf-ocr-layout → Tesseract
- **异常标记自动识别**：↑↓△H L 阳性 偏高 偏低 BI-RADS/TI-RADS分级≥3
- **终检结论**是核保评估最核心的依据，务必完整提取
- **多次报告对比**：当存在历史体检报告时，必须对比关键指标趋势（恶化/好转/新发/消失）

**5. 门诊/住院病历**
- 关注确诊日期、治疗经过、手术记录、出院诊断、随访状态

### PDF提取技术选型

| 方案 | 优势 | 劣势 | 适用场景 |
|------|------|------|---------|
| PyMuPDF | 速度快、准确、零API | 仅文字型PDF | 瑞慈/美年/爱康等电子PDF |
| PaddleOCR | 中文识别最佳 | 需GPU加速 | 扫描件、图片 |
| pdf-ocr-layout | 多模态深度分析 | 需智谱API | 复杂版式文档 |
| markitdown | 微软官方 | 依赖重、安装常失败 | 不推荐Windows环境 |

### 体检报告结构认知

中国主流体检报告通常包含以下模块（顺序可能不同）：

```
1. 封面/头部：姓名、性别、年龄、体检日期、体检编号
2. 健康问卷：既往史、手术史、外伤史、过敏史
3. 一般检查：身高、体重、BMI、血压
4. 内科/外科/妇科/眼科/耳鼻喉/口腔
5. 检验报告：尿常规、血常规、生化（肝功/肾功/血糖/血脂）、免疫发光（甲功/肿瘤标志物）
6. 影像报告：超声（甲状腺/乳腺/腹部/妇科）、CT、X线
7. 心电图
8. 终检结论（最重要！体检机构综合总结）
9. 影像附件页（10-20页，应跳过不提取）
```

### 核保关键信息提取清单

从体检报告中，以下信息对核保评估最关键：

| 优先级 | 信息类别 | 提取要点 | 核保影响 |
|--------|---------|---------|---------|
| P0 | 终检结论 | 完整提取所有编号结论 | 直接决定核保结论 |
| P0 | 手术史 | 手术名称、时间、距今年限 | 先心术后等需专项评估 |
| P0 | 结节分级 | BI-RADS/TI-RADS/Lung-RADS分级+大小 | 决定除外/延期/标体 |
| P0 | HPV+TCT | 型别、阳性/阴性、TCT分级 | 宫颈癌风险评估 |
| P1 | 血脂四项 | TC/TG/HDL/LDL数值+参考值 | 心血管加费评估 |
| P1 | 血压 | 收缩压/舒张压 | 高血压分级评估 |
| P1 | BMI | 身高/体重/BMI | 体重过低/过高加费 |
| P1 | 甲功 | TSH/FT3/FT4 | 甲状腺疾病评估 |
| P2 | 血常规 | Hb/WBC/PLT | 血液系统筛查 |
| P2 | 肝肾功能 | ALT/AST/Cr/UA | 代谢/器官功能 |
| P2 | 肿瘤标志物 | AFP/CEA/CA系列 | 肿瘤筛查 |
| P2 | 血糖 | FPG | 糖尿病筛查 |

### 多次报告对比核保评估要点

当存在多次体检报告时，核保评估必须关注趋势变化：

| 变化类型 | 核保影响 | 示例（张瑜案例） |
|---------|---------|----------------|
| 增生→结节 | 风险升级，需重新评估 | 乳腺小叶增生→BI-RADS 3类 |
| 正常→超标 | 新增加费/除外因素 | TC 5.45→6.30, LDL 3.36→4.21 |
| 阳性→未复查 | 需补充最新结果 | HPV16/18+2025检出，2026未查 |
| 未告知→新告知 | 需专项评估 | 先心术后2026年新告知 |
| 结节稳定 | 可排除延期 | 肺结节需CT随访确认稳定 |
| 指标好转 | 可缩小除外/降低加费 | TSH 4.15→2.631正常化 |
