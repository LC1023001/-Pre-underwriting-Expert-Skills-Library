# 核保理赔实务参考指南

## 一、常见重疾险保障范围（对应知识库内容）

### 恶性肿瘤类（重点关注）
- 食管癌、胃癌、结直肠癌、肝癌、胰腺癌 → 内科学
- 肺癌 → 内科学 + 肿瘤学
- 乳腺癌、宫颈癌、卵巢癌 → 妇产科学
- 前列腺癌 → 泌尿外科学
- 脑恶性肿瘤 → 神经外科学 + 肿瘤学
- 白血病、淋巴瘤 → 内科学 + 病理学

### 心脏类重疾
- 急性心肌梗死 → 内科学（冠心病章节）
- 严重冠心病（需搭桥/支架）→ 外科学
- 严重心衰（NYHA分级）→ 内科学
- 心脏瓣膜手术 → 外科学

### 脑部重疾
- 严重脑中风（脑梗死/脑出血）→ 神经内科学
- 良性脑肿瘤（需开颅手术）→ 神经外科学
- 帕金森病、阿尔茨海默病 → 神经内科学

### 器官类重疾
- 重大器官移植（肝/肾/心/肺）→ 外科学
- 终末期肾病（尿毒症透析）→ 内科学（肾内科）
- 肝硬化失代偿 → 内科学（消化内科）

### 其他
- 严重烧伤 → 外科学
- 脊髓损伤 → 骨科学 + 神经外科学
- 双耳失聪/双目失明 → 耳鼻咽喉科学 + 眼科学

## 二、核保关注的核心医学指标

| 指标 | 核保关注点 | 知识库来源 |
|------|-----------|-----------|
| 血压 | 高血压分级、靶器官损害 | 内科学心血管 |
| 血糖/HbA1c | 糖尿病分型、并发症 | 内科学内分泌 |
| 肾功能（eGFR/Cr）| CKD分期 | 内科学肾内科 |
| 肝功能（ALT/AST）| 肝炎/肝硬化 | 内科学消化 |
| 心电图 | 心律/缺血改变 | 诊断学 |
| 影像学（CT/MRI）| 肿瘤/结节分级 | 肿瘤学+影像学 |
| 肿瘤标志物 | AFP/CEA/CA199等 | 实验诊断学 |
| 病理报告 | 良恶性/分级/分期 | 病理学 |

## 三、理赔审核关键点（从医学知识库提取）

### 诊断真实性
- 疾病定义与实际诊断是否匹配（看病理/影像/实验室依据）
- 疾病分期是否达到重疾标准
- 发病时间是否符合等待期规定

### 既往症排查
- 投保前是否已有相关症状/诊断（对应知识库中的疾病定义）
- 体检异常与主诉是否一致

### 治疗充分性
- 手术方式是否符合重疾定义（如开颅 vs 微创）
- 放化疗方案是否完整执行

## 四、疾病ICD-11速查思路

知识库中内科学等教材已涵盖ICD疾病分类体系，搜索时可加关键词：
- "ICD" + 疾病名
- "分类标准" + 系统名（如心血管疾病分类）
- "诊断标准" + 具体疾病

## 五、重要免责声明

医学知识库内容代表**经典医学知识**，核保/理赔**最终结论**必须：
1. 严格对照具体保险产品条款
2. 结合被保险人完整病史
3. 依据投保时健康告知情况
4. 参考当时有效法律法规及监管规定
