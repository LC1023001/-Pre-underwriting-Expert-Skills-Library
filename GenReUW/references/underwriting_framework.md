## 一、核保决策树

```
投保申请
    │
    ├─ 无既往症/异常 ──→ 标准体承保
    │
    └─ 有健康异常
           │
           ├─ 评估严重程度
           │      │
           │      ├─ 轻度/已治愈 ──→ 标准体 or 加费
           │      ├─ 中度/稳定  ──→ 加费 or 除外
           │      ├─ 重度/活动期 ──→ 延期 or 拒保
           │      └─ 不可预测   ──→ 拒保
           │
           └─ 评估可保性要素
                  ├─ 诊断明确度
                  ├─ 治疗依从性
                  ├─ 预后/存活率
                  ├─ 末次发作/治疗时间
                  └─ 并发症/合并症
```

## 二、主要系统核保要点速查

### 循环系统（Cardiovascular）

| 疾病 | 关键核保因素 | 寿险参考结论 | DI参考结论 |
|------|-------------|------------|-----------|
| 高血压 | BP水平、靶器官损害、治疗达标情况 | 轻度达标→标准；重度→+50-200%EM | 轻度→标准；重度→加费/除外 |
| 心房颤动 | CHA₂DS₂-VASc评分、类型（孤立/结构性）、抗凝治疗 | 孤立性→+50%EM；结构性→+100-200%EM | 严格加费 |
| 冠心病/心绞痛 | 血管病变程度、LVEF%、血运重建时间 | 稳定型+PTCA≥2年→+50-100%EM | 多数除外/拒保 |
| 心肌梗死 | 距今时间、LVEF%、残余缺血 | ≥5年+EF>50%→+100-200%EM | 拒保 |
| 心肌病（扩张型）| LVEF%、心功能分级 | EF<40%→拒保 | 拒保 |
| Brugada综合征 | 有无症状（晕厥）、ICD植入否 | 无症状→+200%EM；有症状→拒保 | 拒保 |
| 主动脉瓣狭窄 | 狭窄程度（轻/中/重）、手术否 | 轻度→+50%EM；重度→延期或拒保 | 加费/拒保 |

### 肿瘤（Oncology）

| 肿瘤类型 | 关键核保因素 | 寿险参考结论 |
|---------|-------------|------------|
| 基底细胞癌 | 治疗情况 | 完全切除→标准体 |
| 甲状腺乳头状癌 | 分期、治疗情况 | I期治愈后→+50%EM（5年后） |
| 乳腺癌 | 分期、受体状态、手术时间 | I期≥5年→+100%EM；II期≥5年→+200%EM |
| 结肠癌 | 分期、手术时间 | Dukes A≥5年→+50%EM |
| 前列腺癌 | Gleason评分、PSA趋势 | 低级别局限性→+50-100%EM |
| 肺癌 | 分期、病理类型 | I期≥5年→+200%EM；II期以上→拒保 |
| 黑色素瘤 | Clark分级 | Level I/II→标准体；Level IV/V→拒保 |
| 宫颈癌（CIN） | CIN级别 | CIN1→标准体；CIN3切除后→+50%EM |

### 神经系统（Neurology）

| 疾病 | 关键核保因素 | 寿险 | DI |
|------|------------|------|-----|
| 癫痫 | 发作频率、控制情况、病因 | 良控≥2年→+50%EM；不良控→+200%EM | 多数除外/拒保 |
| 帕金森病 | 进展期、功能状态 | 早期→+100%EM；进展期→拒保 | 拒保 |
| 脑卒中 | 距今时间、残余缺陷、病因 | 缺血性≥2年无残→+100%EM | 多数拒保 |
| 多发性硬化 | 复发次数、EDSS评分 | 缓解型→+200%EM；进展型→拒保 | 拒保 |
| 偏头痛 | 频率、有无先兆 | 有先兆→+25-50%EM | 轻度→标准体 |

### 精神和行为障碍（Psychiatry）

| 疾病 | 关键因素 | 寿险 | DI |
|------|---------|------|-----|
| 轻度抑郁（单次）| 康复时间、自杀史 | 康复≥2年→标准体 | 标准体 |
| 复发性抑郁 | 发作次数、治疗情况 | +50-150%EM | 除外/加费 |
| 双相情感障碍 | 发作频率、治疗依从性 | +200%EM | 拒保 |
| 精神分裂症 | 活动期/缓解期 | 活动期→拒保 | 拒保 |
| 酒精依赖 | 戒断时间（≥2年）、肝脏损害 | 戒断≥5年→+100%EM | 除外 |
| 焦虑障碍（轻度）| 治疗情况 | 轻度→标准体 | 标准体 |
| 进食障碍 | 活动期/康复 | 活动期→拒保 | 除外/拒保 |

### 内分泌系统（Endocrinology）

| 疾病 | 关键因素 | 核保参考 |
|------|---------|---------|
| 2型糖尿病 | HbA1c、并发症、BMI | HbA1c<7%→+50%EM；并发症→+100-300%EM |
| 1型糖尿病 | 发病年龄、控制、并发症 | 多数+100-200%EM |
| 甲状腺功能亢进 | 治疗情况、眼病 | 治疗后稳定→标准体 |
| 甲状腺功能减退 | 治疗后TSH正常否 | 治疗稳定→标准体 |
| 肥胖 | BMI分级 | BMI 30-35→+25%EM；>40→+100%EM |

## 四、财务核保原则

1. **收入替代率原则**：DI保额不超过税前收入75%（扣除已有保障）
2. **道德风险控制**：总保额（所有保司合计）不得超过年收入的×10倍（寿险经验值）
3. **核实收入文件**：报税单、薪资单、营业执照（自雇）
4. **特殊职业限额**：部分高危职业有最高保额限制

## 七、核保常用公式与计算

### CVD心血管风险评分（Gen Re计算器因素）
- 年龄、性别
- 收缩压
- 总胆固醇 / HDL
- 吸烟状态
- 糖尿病
- 家族史

### BMI计算
```
BMI = 体重(kg) / [身高(m)]²
正常: 18.5-24.9  超重: 25-29.9  肥胖I: 30-34.9  肥胖II: 35-39.9  肥胖III: ≥40
```

### 常用加费对照参考
| 加费幅度 | 对应额外死亡率 | 典型疾病 |
|---------|-------------|---------|
| +25%EM | 轻微增加 | 轻度高血压、轻度超重 |
| +50%EM | 中度增加 | 控制良好的糖尿病、孤立性房颤 |
| +100%EM | 明显增加 | 冠心病稳定期、中度肥胖 |
| +200%EM | 显著增加 | 多次心梗、多发性硬化 |
| +300%EM | 极高风险 | 严重心肌病、进展期癌症 |
