## 核保评点标准框架

Gen Re CLUE 对每种疾病提供三条产品线的独立评点：

### 1. 寿险（Life）
- **标准体承保**：预后良好、风险可控
- **加费承保**：Extra mortality，通常以 `+50%EM` / `+100%EM` 等表示
- **延期承保**：需观察期后重新评估
- **除外承保**：特定病因/部位除外
- **拒保**：风险超出承保范围

### 2. 收入保障险（Income Protection / DI）
- 对慢性病、精神疾病、肌骨疾病更为严格
- 常见结论：职业别除外、部分除外、加费、拒保
- 关注：发作频率、治疗依从性、职业适应性

### 3. 健康险（Health / CI）
- 重疾险关注：疾病定义符合度、存活率、复发率
- 医疗险关注：慢性病持续花费、既往症
- 常见结论：带病承保、除外、加费

## 核心评点逻辑（深度学习要点）

### 心血管系统核保要点
1. **心房颤动**：需看 CHA₂DS₂-VASc 评分；孤立性房颤（无器质性病变）可标准承保；持续性+抗凝治疗→加费
2. **冠心病**：看最后一次发作距今时间、血运重建情况（CABG vs PTCA）、射血分数（EF%）
3. **Brugada综合征**：有症状（晕厥/VF史）→拒保；无症状ECG异常→高额加费；已植入ICD→可评估
4. **高血压**：需看靶器官损害；单纯高血压（BP<160/100且治疗达标）→标准体；并发症→加费
5. **心肌病**：扩张型心肌病 EF<40%→拒保；肥厚型+猝死家族史→拒保

### 肿瘤核保要点
1. **等待期原则**：多数肿瘤需要5年无复发观察期
2. **分期影响**：I期肿瘤康复后可能标准承保；III/IV期多数拒保或高额加费
3. **特殊情形**：基底细胞癌（低风险，5年后可标准体）；甲状腺癌（乳头状，I期治疗后预后良好）
4. **肿瘤标志物异常**：AFP↑、CEA↑、CA125↑需进一步确认是否有恶性病变

### 精神与行为障碍核保要点
1. **抑郁症**：单次轻度发作、已康复→标准体；复发性→加费或除外
2. **双相障碍**：寿险加费；收入保障险多数拒保
3. **药物/酒精依赖**：戒断≥2年且无复发→加费承保；现患→拒保
4. **焦虑障碍**：轻度→标准体；重度/慢性→加费

### 糖尿病核保要点
1. **1型糖尿病**：发病年龄、并发症（视网膜病、肾病、神经病变）决定评点
2. **2型糖尿病**：HbA1c控制水平（<7%标准、7-9%加费、>9%拒保参考）
3. **BMI**：肥胖（BMI>30）对寿险和DI均有影响

### 职业风险核保要点
1. **Gen Re 职业分类**：参照计算器，分1-5级
2. **特殊职业**：商业潜水、商业航空、军队→拒保DI
3. **职业运动员**：特定保额限制

### 业余爱好风险评点要点
1. **水下潜水**：超过40米深度→加费或除外
2. **赛车（汽车/摩托车）**：业余赛车→加费；职业赛车→拒保
3. **跳伞**：不超过特定次数→加费；职业→拒保
4. **登山**：8000米以上→拒保

## 核保查询工作流程

当用户询问某疾病核保问题时，按以下步骤处理：

### Step 1：识别疾病/风险类型
- 提取疾病名称 → 匹配 ICD-10 代码
- 确认所属系统（循环/肿瘤/神经等）

### Step 2：调用知识库搜索
使用脚本 `scripts/search_disease.py` 在知识库中精确检索：
```
python "C:\Users\lucha\.workbuddy\skills\GenReUW\scripts\search_disease.py" --query "疾病名" [--system 系统名]
```

### Step 3：提取评点要素
从搜索结果中提取：
- **ICD代码**
- **病因与临床表现**
- **核保要求（APS项目）**
- **寿险评点**
- **收入保障险评点**
- **健康险评点**
- **末次更新年份**

### Step 4：结合中国市场实际给出建议
Gen Re CLUE 基于欧洲/国际市场，应用时需注意：
- 中国寿险市场不销售收入保障险（Income Protection）
- 重疾险适用健康险评点
- 保额标准参照中国市场实务（非欧洲标准）
- 等待期要求可能有所不同

### Step 5：输出格式
```
## [疾病名] 核保评点（Gen Re CLUE）

**ICD-10：** Xxx.x
**系统分类：** xxx系统
**末次更新：** 20xx年

### 核保要求（APS）
- 项目1
- 项目2

### 寿险评点
[具体评点规则]

### 收入保障险评点（参考）
[具体评点规则]

### 健康险/重疾险评点
[具体评点规则]

### 中国市场应用建议
[基于中国核保实务的补充说明]
```

## 快速参考：常见核保结论代码

| 代码 | 含义 | 说明 |
|------|------|------|
| SER | 标准体承保 | Standard Extra Risk = 0 |
| +EM | 加费 | Extra Mortality，如 +50%EM |
| +EP | 加保费 | Extra Premium |
| DEF | 延期 | Deferral，观察一段时间后重评 |
| EXC | 除外承保 | Exclusion，特定病因/部位除外 |
| DEC | 拒保 | Decline |
| TPD | 全残 | Total Permanent Disability 相关条款 |
