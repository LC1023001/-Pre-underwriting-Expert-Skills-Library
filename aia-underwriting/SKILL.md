# AIA友邦保险核保规则 Skill

## 概述

这是一个专业的友邦保险(AIA)核保规则专家系统，基于友邦中国2019年6月发布的《AIA Company, Limited 中国区标准核保规则》编制。

## 核心能力

### 1. 寿险核保规则

#### 投保限额
- 每份寿险投保单主合同最低保额：具体参见产品核保规则
- 累积寿险最高保额：根据财务核保及临分再保险结果而定
- 重疾险产品最高保额：根据财务核保及临分再保险结果而定

#### 保险费最低限额
- 每份寿险投保单（含所有附加合同）每次支付保险费最低限额：**200元**
- 投连及万能保险的追加保险费最低：**2,000元**
- 月付保险费最低限额：**200元**（首期须为二个月保费之和）

#### 付费方式
- 可选择：**年付、半年付、季付或月付**
- 月付只限银行自动转账
- 保险费计算：
  - 半年付 = 年保险费 × 0.51
  - 季付 = 年保险费 × 0.27
  - 月付 = 年保险费 × 0.09

#### 累计免体检限额规则
- 生效/复效时间≥6个月的保单：**不累计入体检保额**
- 生效/复效时间<6个月的保单：**按100%累计**
- 所有累计寿险及重疾险保额≥600万：**仍需体检**（不含高净值业务产品）
- 此规则仅适用于：既往标准体承保（无异常）、投保年龄≤59岁

### 3. 未成年人投保规定

根据《中国保监会关于父母为其未成年子女投保以死亡为给付保险金条件人身保险有关问题的通知》（保监发〔2015〕90号）：

| 被保险人年龄 | 累计死亡保额限制 |
|------------|----------------|
| 不满10周岁 | 不超过**20万元** |
| 已满10周岁但未满18周岁 | 不超过**50万元** |

**可不计入限额的情况：**
1. 投保人已交保险费或被保险人死亡时合同的现金价值/账户价值
2. 合同约定的航空意外死亡保险金额
3. 合同约定的重大自然灾害意外死亡保险金额

**其他规定：**
- 17岁或以下未成年人投保，投保人须为父母一方或法定监护人
- 未成年人重疾累计风险保额超过**100万**，需安排临分再保险

### 4. 特殊人群投保规定

#### 家庭主妇、学生(≥18岁)
- ADD最高限额：**30万**
- 重疾险最高限额：**30万**
- 医疗险：可申请

#### 退休人士
- ADD最高限额：**30万**
- 重疾险最高限额：**30万**
- 家庭主妇年龄≥50岁：按退休人员分类

#### 待业者
- **ADD、HI/ICU、HR/SB均不予承保**

#### 怀孕妇女
- 怀孕不足4个月：正常保单处理
- 怀孕4-7月：需提供产前检查报告
- 怀孕7个月以上：**延期至产后再投保**
- **任何怀孕妇女HI/ICU、HR/SB、康逸、康惠、传世无忧等不予承保**

### 5. 体检要求

#### 体检结果有效期（1年）
满足以下条件，体检结果有效期为1年：
1. 体检后以标准体承保，或加费≤+50%em
2. 无新增的健康状况异常
3. 新申请保单的保额不超过原体检单的保额
4. 客户所有累计寿险及重疾险保额低于600万

#### 体检费承担规则
- 公司承担：按公司要求体检时
- **营销员佣金扣除**：
  - 体检正常通过后，客户申请撤销保单或降低保额至非体检件者
  - 非体检件而自行作体检者
  - 体检后不投保者

#### 可免除体检的情况
1. 主合同缴费方式为趸缴，保费大于身故保险金额
2. 主合同分期缴付，首期保费大于身故保险金额，且未附加WP等其它附加合同
3. 特殊产品（金福、今世无忧等）月付年金低于2000元，定期寿险累计20万以下

### 6. 意外险核保规则

#### 意外险产品主要类型
- **CIPA**：友邦安益综合意外伤害保险计划
- **3YADD II**：友邦附加添益II意外伤害保险计划

#### 最低保险金额
| 产品 | 一类地区 | 二类地区 |
|-----|---------|---------|
| ADD (CIPA) | 10万元 | 6万元 |
| ADD (3YADD II) | 8万元 | 5万元 |

#### 个人最高累计投保限额
- **ADD**：累积最高保额由财务核保结果而定
- **AMR**：ADD保险金额的20%，最高为15万
- **HI/ICU**：30份/人
- **SB/HR**：30份/人

#### 搭配规则
- 每2万ADD最高可附加1份HI&ICU
- 每2万ADD最高可附加1份SB&HR
- 累计ADD保额达**100万**或以上需人工核保

### 7. 健康险核保规则

#### 免体检限额
参阅寿险免体检限额

#### 最高累计保险金额
- 重疾险产品的最高保额根据临分再保险结果而定

#### 医学核保非标体处理原则
对于医疗险，原则上：
- **建议拒保**：评估后MR-F，或有5个以上除外，或MR-D以上+除外
- 其他情况由核保员根据具体情况评估决定

### 8. 财务核保要求

#### 最高保险金额计算
根据被保险人的年收入计算最高寿险、意外险、重疾险保险金额：
- 最高保险金额包括该客户在本公司及其他公司的所有新申请和现时有效之保险金额
- 收入是指职业或工作赚得的收入，不含利息、股息分红、房租等投资来源收入

#### 保险费财务核保
- **一般规则**：年保险费不应超过投保人年收入的**30%**
- **趸缴/Top-up保险费**（每笔≥50万）：需人工审核缴纳保险费能力

#### 高保额保险单所需资料
1. **客户财务背景报告**（营销员填写）：
   - 客户投保目的
   - 承揽经过
   - 客户家庭及社会背景情况
   - 客户及其家庭成员财务状况
   - 家庭成员保障情况
   - 客户日常生活习惯及业余爱好

2. **有效财务证明**：
   - 薪金证明、雇佣合同
   - 经审计之公司帐目、损益表及经营执照
   - 完税凭证
   - 银行存款证明
   - 房地产证明
   - 个人名下资产证明

### 9. 反洗钱(AML)规则

#### 客户身份识别要求

**保险费计算规则：**
- 保险费 = 趸缴保险费 + 期交保险费(首期保费×总缴费期数) + 追加保险费
- 1年期可续保产品，续保保费勿需累计
- 终身付费产品在投保时均要提供客户身份证件复印件

**有效身份证件：**
| 人员类型 | 有效证件 |
|---------|---------|
| 16岁以上中国公民 | 居民身份证 |
| 中国人民解放军军人 | 军人身份证 |
| 香港、澳门居民 | 港澳居民往来内地通行证、港澳台居民居住证 |
| 台湾居民 | 台湾居民来往大陆通行证、港澳台居民居住证 |
| 外国公民 | 护照 |

#### PEP筛查
- 年化保险费**50万（含）人民币以上**的保单需进行PEP筛查
- 通过AML系统对投保人、被保险人及受益人进行查询和确认

### 10. 核保决策矩阵

#### 承保结论
| 结论 | 说明 |
|-----|-----|
| **标准体承保** | 无异常健康状况 |
| **加费承保** | 特定风险因素，需额外收取保险费 |
| **除外责任** | 特定疾病或部位不予保障 |
| **延期** | 暂不接受投保，需观察后续情况 |
| **拒保** | 不接受投保申请 |

#### 次标准保单处理流程
1. 核保部发照会并附'修订计划'或'不保事项'
2. 客户签名同意
3. 如需加费则补交追加之保险费
4. 客户健康状况改善后可申请重审（生效第一个周年日后）

## 使用场景

1. **投保咨询**：根据客户年龄、职业、健康状况、财务状况提供核保建议
2. **核保评估**：评估投保申请是否符合友邦核保规则
3. **产品搭配**：确定各产品间的搭配比例和最高限额
4. **非标体处理**：评估次标准体的承保条件和修订计划
5. **反洗钱合规**：确认客户身份识别和资金来源证明要求
