## 一、核保专家技能体系总览

本技能库围绕健康险核保全流程构建，涵盖5个核心专家技能，形成从接单→OCR识别→核保评估→理赔审核的完整闭环。

### 1.1 技能矩阵

| 技能名称 | 版本 | 核心定位 | 知识来源 | 条目覆盖 |
|---------|------|---------|---------|---------|
| **ci-underwriting** | v2.6+ | 重疾险核保评估 | 慕再MIRA/NRS + 中国保司规范 | 10大系统200+疾病 |
| **ci-claim-review** | v1.0 | 重疾理赔审核与调查 | 行业理赔实务 + 欺诈特征库 | 5阶段审核框架 |
| **aia-underwriting** | v1.0 | 友邦保险核保规则 | AIA中国区标准核保规则(2019.6) | 寿险/重疾/意外/健康 |
| **GenReUW** | v1.0 | Gen Re CLUE核保手册 | 通用再保险CLUE系统 | 1,159个疾病/条目 |
| **wechat-underwriting-intake** | v3.0 | 微信核保接单预处理 | 实战案例(张瑜等) | 5种输入格式 |

## 二、核保案例测试经验汇总

### 2.1 案例一：张瑜 — 多次体检报告对比核保评估

**案例背景：** 女性，提供两份瑞慈体检报告（2025-01-18 vs 2026-01-02），PDF文字型。

**测试过程：**

1. **PDF提取方案选型**
   - 首次尝试：`markitdown[pdf]` → Python 3.13 安装依赖冲突（失败）
   - 二次尝试：`pdfplumber` → Windows Python 3.14 pip install 超时（失败）
   - 最终方案：**PyMuPDF (fitz)** → 毫秒级提取，准确率极高（成功）

   > **经验沉淀：** Windows环境PDF提取，PyMuPDF是首选。安装轻量（`pip install PyMuPDF`），零API依赖，中文医学文本提取完整。

2. **体检报告结构识别**
   - 瑞慈体检PDF共20+页，后10-20页为影像附件页（无实质文字内容）
   - **解决：** 跳过 `text.strip().startswith("影像附件")` 的页面
   - 检验项目跨页断裂（如血常规表格被分页符切断）
   - **解决：** 逐页提取后按"项目名称/结果/参考值"模式拼接

3. **关键核保风险发现**

   | 异常类型 | 报告1(2025) | 报告2(2026) | 变化趋势 | 核保影响 |
   |---------|------------|------------|---------|---------|
   | 乳腺 | 双侧小叶增生 | 左乳结节BI-RADS 3类 | ⬆进展 | +25，延期6月或除外乳腺 |
   | 血脂TC | 5.45↑ | 6.30↑ | ⬆恶化 | 加费因子 |
   | 血脂LDL-C | 3.36 | 4.21↑ | ⬆恶化 | 加费因子 |
   | HPV | 未检测 | HPV16/18阳性 | 新发 | 需TCT结果 |
   | 先心 | 未告知 | 动脉导管修补术30年 | 新告知 | 需专项评估 |
   | 甲功TSH | 4.15 | 2.631 | ✅正常 | 风险消除 |

4. **核保结论：** 加费+除外（乳腺相关重疾+心血管相关重疾），综合评点约+75~+100

**关键经验：**
- **多次报告对比是核保评估的核心**：单次报告只能看到静态风险，对比才能发现进展/恶化/新发
- **终检结论是核保最核心依据**：体检报告的终检总结优先级最高
- **参考值含性别条件**：如"男:9-50U/L 女:7-40U/L"，需根据被保人性别匹配
- **TSH参考值含孕期**：使用"正常人群"参考范围

### 2.2 案例二：乳腺结节核保评估

**测试场景：** 通过ci-underwriting技能评估不同BI-RADS分级的核保结论

| BI-RADS分级 | 核保评点 | 核保结论 | 关键条件 |
|-------------|---------|---------|---------|
| 1-2级（良性） | 0 | 标准体 | — |
| 3级 | +25，延期随访 | 6月随访后重评 | 需超声随访确认稳定 |
| 4-5级 | 延期/Dec | 需活检明确性质 | 活检结果决定最终结论 |
| 乳腺癌(任何) | Dec | 拒保 | — |
| BRCA1/2突变 | +100~Dec | 个案高度关注 | 家族史+基因检测 |

**关键经验：**
- BI-RADS 3级是核保分水岭：3级以下通常标体，3级开始进入加费/除外区间
- 乳腺增生→结节的进展需关注时间跨度
- BRCA基因突变携带者的核保处理需要个案评估

### 2.3 案例三：乙肝携带者核保评估（东亚重点）

**测试场景：** 不同HBV活动状态的核保评点对比

| 状态 | HBV DNA | ALT | 评点 | 策略 |
|------|---------|-----|------|------|
| 表面抗体(+), 抗原(-) | 阴性 | 正常 | 0 | 标准体（已免疫） |
| 乙肝携带者(非活动) | <2000 IU/mL | 正常 | +25~+50 | 轻加费+肝病除外 |
| 慢性乙肝(轻度活动) | 2000~20000 | 轻度升高 | +50~+100 | 加费+肝脏除外 |
| 慢性乙肝(中重度) | >20000 | 明显升高 | +100~+200 | 高额加费或拒保 |
| 肝硬化 | — | — | Dec | 拒保 |

**东亚修正因子：** 乙型肝炎风险权重 ×1.5（中国HBV感染率远高于欧美）

**关键经验：**
- HBsAg+在中国投保人中比例高，是东亚人群核保重点
- 核保关注三要素：HBV DNA载量 + ALT水平 + 肝纤维化程度
- 非活动携带者（DNA<2000, ALT正常）仍需加费+肝病除外

### 2.4 案例四：甲状腺结节核保评估

**测试场景：** TI-RADS分级核保评点

| TI-RADS分级 | 评点 | 核保结论 | 备注 |
|-------------|------|---------|------|
| 1-2级（良性） | 0 | 标准体 | — |
| 3级 | +25, 延期随访 | 6月随访后重评 | 需超声随访 |
| 4-5级 | 延期/Dec | 需活检明确性质 | FNA细针穿刺 |
| 乳头状癌术后<5年 | Dec | 拒保 | — |
| 乳头状癌术后>5年(低风险) | +50~+100 | 加费+甲状腺除外 | 预后最好的癌种之一 |

**关键经验：**
- 甲状腺乳头状癌是"最友好的癌症"，术后5年以上可承保
- 甲亢活动期需延期，甲减服药控制TSH正常基本标体
- TI-RADS 3级与BI-RADS 3级处理逻辑一致

### 2.5 案例五：高血压+糖尿病组合（代谢综合征）

**测试场景：** 多种代谢异常组合的关联折叠评点

```
案例：男性52岁，BMI 26.5，1级高血压(药物控制)，2型糖尿病(HbA1c 7.2%)

单项评点：
  BMI 26.5（超重）:          +25
  1级高血压(控制达标):        +50
  2型糖尿病(HbA1c 7.2%):     +100

相关性折叠计算：
  代谢综合征（肥胖+高血压+糖尿病）→ 最高项(+100) + 次高项×0.5(+50×0.5=+25) + 第三项×0.3(+25×0.3=+7.5)
  综合评点 ≈ +132

核保结论：加费+除外（糖尿病相关重疾+心血管相关重疾）
```

**关键经验：**
- **相关性折叠原则**是处理代谢综合征等共病的关键
- 同一病理机制（代谢综合征：肥胖+高血压+糖尿病）不简单叠加
- 重疾险除外责任上限建议不超过3项
- 综合评点超过400%自动进入经验核保（个案评估）

### 3.4 ci-claim-review 五阶段框架定型

从零构建理赔审核标准框架：
1. **阶段一**：基本资格审查（保单效力+等待期+投保节点逆向排查）
2. **阶段二**：诊断合规性审查（机构资质+诊断标准+一致性）
3. **阶段三**：医疗资料真实性鉴别（文书审查+医疗逻辑+跨机构核查）
4. **阶段四**：调查策略决策（发起标准+4类调查方式+调查函模板）
5. **阶段五**：理赔结论形成（5种结论类型+结构化输出格式）

**独创风险因子评分系统：** 极高风险（4项）+ 高风险（3项）+ 中风险（2项）+ 低风险（1项），累计≥2高风险因子或任一极高风险→建议发起调查

## 五、核保评估核心方法论

### 5.1 NRS数字评点法（改良版）

```
评点基准：标准体死亡率/发病率为100%

重大疾病险评点决策标准：
┌──────────────────────────────────────────┐
│ 0点         → 标准体（标准保费承保）         │
│ +25~+75点   → 轻度加费（+25%~+75%额外保费）  │
│ +100点      → 中度加费（双倍保费）           │
│ +100+除外   → 组合条件承保                  │
│ +200点以上   → 高度加费或拒保               │
│ 延期(PP)    → 需观察后重新评估              │
│ 拒保(Dec)   → 风险过高，无法承保             │
└──────────────────────────────────────────┘
```

### 5.2 东亚人群特异性风险修正

| 疾病类型 | 修正系数 | 原因 |
|---------|---------|------|
| 乙型肝炎(HBV) | ×1.5 | 中国HBV感染率5-7% |
| 肝细胞癌 | ×1.8 | 与HBV高度相关 |
| 鼻咽癌 | ×2.0 | 华南/东南亚高发(EBV) |
| 胃癌 | ×1.5 | Hp感染+饮食习惯 |
| 食管癌 | ×1.4 | 华北/中原地区高发 |
| 前列腺癌 | ×0.5 | 东亚发病率低 |
| 黑色素瘤 | 宽松 | 东亚发病率低 |

### 5.3 综合评分关键原则

1. **相关性折叠原则**：代谢综合征（肥胖+高血压+糖尿病）不简单叠加
2. **除外责任上限**：重疾险通常不超过3项除外
3. **评点上限**：综合评点超400%→经验核保
4. **10年心血管风险**：Framingham亚裔修正版，<10%可标体，≥20%考虑加费

## 七、技能使用指南

### 快速上手

1. **核保评估**：使用 `ci-underwriting` 技能，提供体检报告/病历资料
2. **理赔审核**：使用 `ci-claim-review` 技能，提供理赔申请资料
3. **友邦规则**：使用 `aia-underwriting` 技能，查询AIA特定核保规则
4. **Gen Re查询**：使用 `GenReUW` 技能，查询CLUE核保手册评点
5. **微信接单**：使用 `wechat-underwriting-intake` 技能，处理微信发来的客户资料

### 免责声明

本技能库仅提供核保辅助评估，基于公开发布的核保规范和医学证据。实际承保决策须：
1. 由持牌核保人员依据其所在保险公司具体核保规程作出
2. 以保险公司原始核保手册为准（各公司核保尺度存在差异）
3. 符合《保险法》及金融监管总局相关监管规定
4. 重疾险核保涉及健康信息，须严格遵守数据隐私保护法规
