身份证号码：_______________（末四位：XXXX）
保险单号：_______________

✅ 跨机构核查：各医院记录相互印证，无矛盾

保险公司按保险金额给付重大疾病保险金。

处理建议：完成内部流程审批后，向申请人银行账户（尾号_____）划付理赔款。

备注：___________

审核人员：_______________ 
审核日期：_______________

## 快速风险标注图例说明

在审核意见中，使用以下标注符号：

| 符号 | 含义 |
|-----|------|
| ✅ | 核查通过，无疑点 |
| ⚠️ | 存在疑点，需关注 |
| ❌ | 核查不通过，存在明确问题 |
| 🔴 | 极高风险，必须调查 |
| 🟠 | 高风险，建议调查 |
| 🟡 | 中风险，需补充核查 |
| 🟢 | 低风险，正常处理 |
| ⏸ | 延期处理中 |
| ❓ | 信息不足，待核实 |

## 审核意见快速摘要格式（内部流转用）

```
【案件快速摘要】
被保险人：___ | 病种：___ | 申请保额：___万元
风险等级：🟢低 / 🟡中 / 🟠高 / 🔴极高
核心疑点：（一句话描述）
建议结论：✅正常给付 / ⚠️有条件给付 / ⏸调查延期 / ❌拒付
紧急程度：一般 / 加急（保额超敏感阈值或有投诉风险）
审核耗时建议：标准 / 调查（预计___工作日）
```
