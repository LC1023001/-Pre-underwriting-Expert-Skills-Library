# 常见重大疾病病种专项审核标准指引

> 本指引基于中国保险行业协会《重大疾病保险的疾病定义使用规范（2020年修订版）》编制。
> 审核时应以保险合同约定的具体条款为准，本指引作为参考标准使用。

---

## 第一类：恶性肿瘤（含部分严重恶性肿瘤）

### 1.1 保险合同标准定义要点

- 经病理学检查结果（组织病理或细胞病理）明确诊断，确诊为恶性肿瘤
- 包括：白血病、恶性淋巴瘤等血液系统恶性肿瘤
- **通常排除范围**（须仔细核查保单条款）：
  - 原位癌（Carcinoma in situ / Tis期）
  - 交界性肿瘤（如交界性卵巢肿瘤）
  - 皮肤基底细胞癌、皮肤鳞状细胞癌（部分条款排除）
  - 恶性黑色素瘤以外的皮肤癌（部分条款排除）
  - 早期前列腺癌（TNM分期T1N0M0，部分条款限制）
  - 慢性淋巴细胞白血病（Rai分期0期，部分条款限制）

### 1.2 核心审核材料

| 材料 | 要求 | 疑点信号 |
|-----|------|---------|
| **病理诊断报告** | 必须有，含病理号、病理科医师签名、医院公章 | 无病理报告→不符合诊断标准；仅细胞学涂片→需评估 |
| 免疫组化报告 | 部分病种需要（如淋巴瘤分型、乳腺癌HER2状态） | 分型诊断不符 |
| CT/MRI/PET-CT | 评估肿瘤分期和范围 | 影像学描述与病理结果矛盾 |
| 肿瘤标志物 | 辅助参考，不作为单独诊断依据 | 结合影像和病理综合评估 |
| 手术记录 | 如进行手术治疗 | 手术范围与病理分期不符 |
| 治疗记录 | 化疗/放疗/靶向/免疫治疗记录 | 诊断明确但无任何治疗记录→高度疑点 |

### 1.3 常见疑点处理

**疑点1：病理报告描述为"高级别上皮内瘤变"**
→ 高级别上皮内瘤变（如CIN III、HGIN）属于癌前病变，通常不认定为恶性肿瘤。需核查是否存在微浸润成分。

**疑点2：甲状腺微小乳头状癌（直径≤1cm）**
→ 部分保险合同对甲状腺癌有特殊限制。需核查保单条款，2020年修订版规范中对直径≤1cm的甲状腺癌有不同处理规定。

**疑点3：前列腺癌PSA异常但无病理活检**
→ PSA升高不能单独诊断前列腺癌，必须有前列腺穿刺活检病理报告。

**疑点4：肿瘤诊断与病史不符（如急性诊断但临床病程提示慢性进展）**
→ 核查投保前是否有肿瘤相关检查异常记录（肿瘤标志物升高、影像学异常）。

## 第二类：急性心肌梗塞

### 2.1 保险合同标准定义要点

确诊须同时满足以下条件：
1. 典型持续性心前区疼痛（或相当症状）
2. 新出现的典型心电图改变（ST段弓背抬高或病理性Q波）
3. 心肌酶谱动态升高（肌钙蛋白TnI/TnT高于参考值上限99百分位数，且有动态变化）
4. 临床确诊为急性心肌梗塞

**保险合同通常排除：**
- 陈旧性心肌梗塞（非急性期诊断）
- 仅有心电图异常无心肌酶升高
- 不稳定型心绞痛

### 2.2 核心审核材料

| 材料 | 要求 | 疑点信号 |
|-----|------|---------|
| **系列心肌酶谱报告** | 需≥2次动态变化（通常入院、6h、12h、24h各一次） | 仅单次检测；肌钙蛋白正常→诊断依据不足 |
| **急性期12导联心电图** | 发病时和系列复查 | 仅有非典型改变；无新发ST抬高或Q波 |
| 超声心动图报告 | 评估心功能和室壁运动 | 心功能正常→与急性心梗严重程度不符 |
| 冠脉造影报告 | 如进行PCI治疗 | 冠脉正常→需重新评估诊断 |
| CCU/ICU住院记录 | 急性期通常需ICU监护 | 无ICU记录且病情描述严重→疑点 |
| 急诊病历 | 首诊记录 | 无急诊记录，直接住院→核查急诊经过 |

### 2.3 常见疑点处理

**疑点1：诊断书写"陈旧性心肌梗塞"但申请急性心肌梗塞理赔**
→ 陈旧性与急性有本质区别，需明确首次急性发作时间，核查是否在保单有效期内且不在等待期。

**疑点2：仅有肌钙蛋白轻度升高（2-3倍正常值）**
→ 需评估是否排除其他引起肌钙蛋白升高的原因（心肌炎、肺栓塞、肾功能不全等）。

**疑点3：心电图仅有T波改变，无Q波**
→ 非ST段抬高型心肌梗塞（NSTEMI）也在保障范围，但须有肌钙蛋白升高，且需核查保单条款对NSTEMI的具体要求。

## 第三类：脑中风（重大脑中风）

### 3.1 保险合同标准定义要点

因脑血管的突发病变引起脑血管出血、栓塞或梗塞，造成神经系统永久性功能障碍，需同时满足：
1. 影像学（CT或MRI）明确诊断
2. 发病持续180天后，仍遗留下列功能障碍之一：
   - 一肢或一肢以上肢体机能完全丧失
   - 语言能力完全丧失
   - 咀嚼吞咽能力完全丧失
   - 自主生活能力完全丧失

**注意：部分保险合同要求神经功能障碍须持续超过180天**

**保险合同通常排除：**
- 短暂性脑缺血发作（TIA，症状<24小时）
- 脑外伤引起的脑出血
- 无神经功能障碍后遗症的脑中风

### 3.2 核心审核材料

| 材料 | 要求 | 疑点信号 |
|-----|------|---------|
| **头颅CT/MRI报告+图像** | 明确梗死/出血灶，注明部位和范围 | 影像学阴性→重新评估诊断 |
| 神经功能评估 | mRS量表或NIHSS评分（发病时和随访时） | 无功能评估→后遗症认定困难 |
| 发病记录 | 首次症状出现时间和性质 | 症状持续时间<24h→可能为TIA |
| 发病后180天随访记录 | 证明功能障碍持续存在 | 早期理赔（<180天）→功能障碍是否持续需验证 |
| 康复治疗记录 | 神经康复过程 | 无康复记录但申报严重后遗症→核实 |

### 3.3 常见疑点处理

**疑点1：发病后未满180天即申请理赔**
→ 确认保单是否有180天等待规定。如有，需等满180天后复查评估功能状态才能理赔。

**疑点2：CT/MRI仅显示腔隙性脑梗死**
→ 腔隙性脑梗死是否造成符合定义的功能障碍需重点评估，通常腔隙性脑梗死后遗症较轻。

## 第四类：慢性肾功能衰竭（终末期肾病）

### 4.1 保险合同标准定义要点

双肾慢性、不可逆性肾功能损害，已达到终末期肾病阶段，须满足以下任一条件：
- 已接受规律性透析治疗（腹膜透析或血液透析）
- 肾小球滤过率（eGFR）< 15 ml/min/1.73m²（连续检测证实）

**保险合同通常排除：**
- 急性肾功能不全（可逆性）
- 药物/毒物引起的一次性肾损害

### 4.2 核心审核材料

| 材料 | 要求 | 疑点信号 |
|-----|------|---------|
| **系列肾功能报告** | 血肌酐（Cr）、尿素氮（BUN）、eGFR，需多次连续记录（≥3个月） | 仅单次检测→无法确认慢性化 |
| 透析记录 | 透析医嘱、透析机记录、护理记录 | 无透析记录但申报已透析→核查 |
| 肾穿刺病理报告 | 如有，明确病因 | 有助于核查病因与病史的一致性 |
| 尿常规报告 | 蛋白尿、血尿历史记录 | 协助判断慢性化程度 |
| 肾脏影像学 | 超声显示双肾萎缩 | 肾脏正常大小→慢性肾病晚期不典型 |

### 4.3 常见疑点处理

**疑点1：肌酐一次性升高，无慢性病程记录**
→ 需鉴别急性肾损伤（AKI）与慢性肾病（CKD），AKI通常可逆，不符合重疾定义。

**疑点2：投保前有糖尿病/高血压病史（已隐瞒）**
→ 糖尿病肾病和高血压肾病是慢性肾衰的最常见原因，须回查既往病史。

## 第五类：冠状动脉搭桥手术

### 5.1 保险合同标准定义要点

- 须实施了开胸冠状动脉搭桥手术
- 手术目的是改善冠状动脉严重病变造成的心肌供血不足
- **注意**：部分保险合同要求为"开胸手术"，不包括微创冠脉搭桥（MIDCAB）或全内镜冠脉搭桥（TECAB）
- **重要排除**：经皮腔内冠状动脉成形术（PTCA，俗称支架手术）不属于冠脉搭桥

### 5.2 核心审核材料

| 材料 | 要求 | 疑点信号 |
|-----|------|---------|
| **术前冠脉造影报告** | 证明需要手术的冠脉病变严重程度 | 冠脉正常或轻度狭窄→手术指征不足 |
| **手术记录** | 注明手术方式（是否开胸）、搭桥支数、使用血管（乳内动脉/大隐静脉） | 仅做导管手术（PTCA/PCI）非搭桥→不符合 |
| 麻醉记录 | 与手术记录配套，注明麻醉方式 | |
| 体外循环记录 | 传统搭桥通常需要体外循环 | 无体外循环记录→核查是否为非停跳搭桥 |
| 心脏手术知情同意书 | 注明手术方式 | |
| 术后ICU记录 | 搭桥手术后通常需ICU监护 | 无ICU记录→核查手术规模 |

## 第六类：重大器官移植术（含造血干细胞移植）

### 6.1 保险合同标准定义要点

- 因器官不可逆性衰竭，接受了他人器官（心、肺、肝、肾、胰腺、小肠、骨髓/造血干细胞）的移植手术
- 须已实施手术（等待期内不赔，已有供体亦须完成手术）
- 自体骨髓/干细胞移植通常不包括在内（须核查条款）

### 6.2 核心审核材料

| 材料 | 要求 |
|-----|------|
| 移植手术记录 | 移植器官类型、供体信息（脑死亡捐献/活体供体） |
| 移植医院资质 | 须为具有器官移植资质的医院 |
| 术后免疫抑制治疗记录 | 抗排斥药物（他克莫司、环孢素等）用药记录 |
| 器官来源合法性证明 | 医院提供的合法来源证明 |
| 术后随访记录 | 器官功能状态评估 |

## 第七类：终末期肝病（肝功能衰竭）

### 7.1 审核要点

- 需明确肝病病因（病毒性肝炎、酒精性肝病、自身免疫性肝病等）
- 核查投保前是否已有肝炎/肝硬化病史（带病投保高风险）
- 重点检查：Child-Pugh评分（达到C级方符合终末期标准）、腹水、肝性脑病、食管静脉曲张破裂出血等并发症
- 核查有无肝癌（肝癌+肝衰竭不能重复理赔同一事件）

## 八、轻症/中症与重症的区分要点

部分保单同时含有轻症险/中症险，需注意区分：

| 情况 | 审核要点 |
|-----|---------|
| 原位癌 → 通常为轻症/特定疾病 | 核查保单是否含轻症保障，按轻症给付 |
| NSTEMI（非ST抬高心梗）→ 部分保单为中症 | 核查条款具体定义 |
| 早期脑中风（无后遗症）→ 部分保单有轻症保障 | 核查轻症定义和给付比例 |
| 不符合重疾标准的冠脉手术（PCI支架）→ 轻症 | 区分重疾（搭桥）与轻症（支架） |

## 附：常见检查指标参考值（审核用）

| 指标 | 正常参考值 | 重疾意义 |
|-----|---------|---------|
| 肌钙蛋白TnI | <0.04 ng/mL（各医院参考值不同） | 急性心梗：动态升高≥3倍 |
| 肌酐 Cr | 44-133 μmol/L | CKD终末期：通常>700 μmol/L |
| eGFR | >60 ml/min/1.73m² | <15为终末期肾病 |
| Child-Pugh评分 | A级:5-6分 B级:7-9分 C级:10-15分 | 终末期肝病：C级 |
| NIHSS卒中评分 | 0分=正常；1-4轻度；5-15中度；>15重度 | 重大脑中风：通常中重度 |
