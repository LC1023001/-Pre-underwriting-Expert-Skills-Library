# 重大疾病险理赔审核要点核查清单

## 一、必要申请资料清单

### 1.1 基础必要资料（所有重疾理赔通用）

| 序号 | 资料名称 | 要求 | 缺少影响 |
|-----|---------|------|---------|
| 1 | 理赔申请书 | 被保险人/受益人签名，信息完整 | 必须补充，否则无法受理 |
| 2 | 保险单正本或保险凭证 | 原件或加盖公章的复印件 | 必须补充 |
| 3 | 被保险人身份证明 | 身份证/户口本，清晰可辨 | 必须补充 |
| 4 | 疾病诊断证明书 | 加盖医院公章，主治医师签名，注明疾病名称、诊断日期 | 核心资料，必须补充 |
| 5 | 相关病历资料 | 门诊病历、住院病历、检查报告等 | 核心资料，必须补充 |
| 6 | 银行账户信息 | 用于理赔款项划付 | 可补充，不影响前期审核 |

### 1.2 住院类重疾额外资料

| 序号 | 资料名称 | 要求 |
|-----|---------|------|
| 7 | 住院病历（完整版） | 包含入院记录、病程记录、医嘱记录、护理记录 |
| 8 | 手术记录（如有） | 手术名称、术式、术中探查所见、主刀医生签名 |
| 9 | 麻醉记录（如有） | 与手术记录配套 |
| 10 | 出院小结 | 入院诊断、住院经过、出院诊断、出院医嘱 |
| 11 | 费用清单 | 明细费用，与病历记录相符 |

### 1.3 特殊病种额外资料

**恶性肿瘤**：
- 病理诊断报告（必须，含病理号、病理医师签名）
- 免疫组化报告（部分病种需要）
- CT/PET-CT/MRI影像学报告
- 肿瘤标志物检查报告

**急性心肌梗塞**：
- 心电图（发病当时及系列复查，12导联）
- 心肌酶谱报告（肌钙蛋白TnI/TnT动态变化，通常需3次以上）
- 超声心动图报告
- 冠脉造影报告（如已做）

**脑中风**：
- 头颅CT/MRI报告（带影像图像更佳）
- 神经功能评估（mRS量表或NIHSS评分）
- 症状描述（首发症状时间、持续时间）

**慢性肾功能衰竭**：
- 肾功能检查报告序列（BUN、肌酐Cr，需多次，连续3个月以上）
- 透析记录（如已透析）
- 肾穿刺病理报告（如有）

**重大器官移植**：
- 手术记录（移植手术）
- 供体来源证明
- 免疫抑制治疗记录

**冠状动脉搭桥术**：
- 冠脉造影报告（术前）
- 手术记录（注明搭桥支数、使用血管材料）
- 体外循环记录（或心脏停跳记录）

## 二、保单基本要素核查清单

### 2.1 保单效力核查

- [ ] 保单是否处于有效状态（未失效、未终止）
- [ ] 投保日期 vs 确诊日期：确认是否在等待期内
  - **等待期计算**：保单生效日 + 等待期天数（通常90天或180天）= 等待期届满日
  - 若确诊日 ≤ 等待期届满日 → **等待期内，不予给付**
  - 注意：等待期因事故致病通常无等待期限制
- [ ] 保费缴纳状态：最后一次缴费日期，有无欠费宽限期问题
- [ ] 保单是否存在抵押贷款（影响给付金额）

### 2.2 保障范围核查

- [ ] 申请理赔的疾病名称是否在保单重疾列表内
- [ ] 保险金额（保额）确认
- [ ] 是否有豁免保费条款（若被保险人为投保人，重疾确诊后保费是否豁免）
- [ ] 是否附加其他险种（如轻症、中症、医疗险），是否需要同步处理
- [ ] 保单约定的诊断医院级别要求

### 2.3 被保险人身份核查

- [ ] 申请理赔的被保险人与保险合同记载是否一致（姓名、身份证号）
- [ ] 受益人与理赔申请人身份是否一致
- [ ] 如被保险人已身故（某些重疾并发死亡），需核查死亡原因是否与申请的重疾相关

## 三、健康告知核查清单（重点防范带病投保）

### 3.1 健康告知书逐项核查

将投保时的健康告知书内容与以下资料对照核查：

- [ ] 投保前3年内住院记录
- [ ] 投保前3年内体检报告（尤其是异常结果）
- [ ] 投保前门诊就诊记录（慢病、专科门诊、长期用药）
- [ ] 社保/医保报销记录（最能体现真实就医历史）
- [ ] 单位体检记录（如有）

### 3.2 高度关注的健康告知漏告情形

| 漏告类型 | 核查方式 |
|---------|---------|
| 高血压病史 | 检查住院病历中"既往史"记载；用药记录 |
| 糖尿病病史 | HbA1c历史值；降糖药用药记录 |
| 恶性肿瘤病史 | 病理报告时间；肿瘤标志物历史值 |
| 冠心病/心脏病史 | 既往心电图异常；用药史（他汀类、阿司匹林） |
| 肾病史 | 既往蛋白尿/血尿记录；肾功能历史值 |
| 肿瘤标志物升高 | 体检报告中的标志物历史趋势 |
| 肝炎/肝硬化史 | 乙肝两对半；肝功能历史值；抗病毒治疗记录 |
| 精神疾病史 | 精神科门诊记录；精神类药物用药史 |

### 3.3 不如实告知认定要点

认定不如实告知（带病投保）需满足：
1. **客观存在**：投保前确实存在相关疾病/症状/治疗
2. **知情可能性**：被保险人应当知道（有就诊记录、体检记录即可推定知情）
3. **告知义务违反**：在健康告知书中未如实填写
4. **因果关系**（争议点）：违规告知的疾病与申请理赔的重疾是否相关（需要谨慎判断）

> **注意**：不如实告知的法律后果取决于投保时间和违规性质：
> - 投保不足2年：保险公司可以解除合同，退还现金价值
> - 投保超过2年（不可抗辩期届满）：需更充分证据，且仅对故意或重大过失的隐瞒有效
> - 实务建议：超过2年案件的不如实告知处理需法律审查

## 五、常见审核盲区提示

1. **原位癌争议**：部分保险合同将原位癌排除在重疾保障范围外，需仔细核对保单条款，避免直接给付。
2. **微创手术认定**：部分保险合同对心脏手术有"开胸"要求，微创/介入手术（如TAVI）可能不符合。
3. **手术指征核查**：某些外科手术（如冠脉搭桥）有严格手术指征，需核查术前冠脉造影是否符合搭桥标准。
4. **"重大"程度判断**：如"重大脑中风"需有神经功能障碍遗留，一过性脑缺血发作（TIA）不在理赔范围内。
5. **境外就医**：境外医疗机构出具的病历和诊断书，需核查是否经公证认证，中文翻译是否准确。
