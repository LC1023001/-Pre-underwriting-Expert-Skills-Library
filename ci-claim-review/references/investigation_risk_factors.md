# 重大疾病险理赔调查疑点风险因子识别库

## 一、风险信号综合评分体系

### 1.1 风险等级定义

| 等级 | 描述 | 处理建议 |
|-----|------|---------|
| **极高风险** | 存在明显欺诈/舞弊特征，高度可疑 | 必须发起调查，暂缓理赔 |
| **高风险** | 多项异常，可疑程度较高 | 建议发起调查，视调查结果处理 |
| **中风险** | 存在疑点但尚不明确 | 要求补充资料，必要时发起针对性核查 |
| **低风险** | 轻微异常，通常可解释 | 书面核查或补充说明即可 |

### 1.2 风险评分规则

**极高风险因子（每项+4分）：**
- 投保后180天内确诊重疾
- 病历存在明显篡改/伪造迹象
- 恶性肿瘤无病理报告支撑
- 经核查发现投保前已有明确相同疾病诊断记录
- 多家保险公司同期大额投保（短期集中投保）

**高风险因子（每项+2分）：**
- 投保前1年内有相关专科就诊记录未申报
- 治疗方案与诊断明显不符
- 多院就诊轨迹异常（无合理解释的跨地区就医）
- 代理人与理赔机构存在关联关系
- 案件超过案值敏感阈值（≥20万元）
- 社保/医保记录与商保申请资料矛盾

**中风险因子（每项+1分）：**
- 住院天数与病情严重程度不匹配
- 同一家庭多人同期申请重疾理赔
- 费用清单与当地医疗收费标准存在明显差异
- 就医时间与工作/生活规律不符
- 主治医师非相关专科医师

**评分解读：**
- 总分 ≥ 5分：极高风险，必须调查
- 总分 3-4分：高风险，建议调查
- 总分 1-2分：中低风险，针对性核查

## 二、医疗资料真实性风险因子详解

### 2.1 病历文书疑点识别

**形式疑点（文书本身）：**

| 疑点特征 | 风险含义 | 核查建议 |
|---------|---------|---------|
| 病历书写字迹不统一（混合多种笔迹） | 可能补写/代写 | 要求原始病历，核查书写医师 |
| 日期时间存在涂改/覆盖 | 可能篡改关键时间节点 | 向医院调取电子病历记录 |
| 入院当天病历过于完整详细 | 可能倒填病历 | 核查护理记录、医嘱执行时间 |
| 多份文件盖章位置异常（章只盖在空白处） | 可能伪造章印 | 向医院直接核查印章真实性 |
| 检验报告纸张、格式与同期其他报告不一致 | 可能伪造检验报告 | 向检验科核查原始数据 |
| 影像学报告仅有文字无图像 | 无法独立核实影像诊断 | 要求提供影像资料原件 |
| 出院小结与病程记录内容矛盾 | 记录不一致 | 逐段对照核查 |

**内容疑点（医疗逻辑）：**

| 疑点特征 | 风险含义 | 核查建议 |
|---------|---------|---------|
| 诊断重疾但住院<5天（需手术的情况） | 治疗强度不符 | 核查手术记录、麻醉记录 |
| 肿瘤患者无化疗/放疗/靶向/免疫治疗记录 | 诊断可能存疑 | 核查肿瘤科治疗记录 |
| 急性心梗无ICU/CCU记录 | 病情严重程度不符 | 核查入院救治流程 |
| 心肌酶谱仅一次检测 | 不符合诊断规范（需动态监测） | 要求提供系列检测报告 |
| 肾功能衰竭无透析记录且肌酐未达标 | 诊断标准可疑 | 核查肾功能系列指标 |
| 重大手术无输血记录（通常需要） | 手术真实性存疑 | 核查手术记录和输血科记录 |
| 术后无抗感染/营养支持医嘱 | 治疗记录不完整 | 补充医嘱记录 |

### 2.2 诊断标准疑点识别

**恶性肿瘤特殊疑点：**
- 仅有细胞学涂片报告，无病理组织学报告：细胞学报告不能单独支持恶性肿瘤诊断
- 报告为"高度可疑恶性"或"倾向恶性"：未达到明确病理诊断
- 病理报告显示"原位癌"：需核查保单条款，通常不在重疾保障范围
- 低度恶性/交界性肿瘤（如交界性卵巢肿瘤）：需核查条款定义
- 皮肤癌（除恶性黑色素瘤外）：通常不在重疾保障范围
- TNM分期为Tis（原位）：属于原位癌

**心肌梗塞特殊疑点：**
- 仅有ST段抬高，无心肌酶谱升高：诊断不完整
- 肌钙蛋白正常或仅轻度升高：与急性心梗诊断不符
- 无持续胸痛症状记录：需核查主诉描述
- 冠脉造影显示冠脉正常：与急性心梗诊断矛盾

**脑中风特殊疑点：**
- 影像学无明确梗死/出血灶：诊断依据不足
- 症状持续时间<24小时（可能为TIA）：不符合脑中风定义
- 无神经功能缺损后遗症（保险合同可能要求）：需核查功能评估
- 发病原因为颅脑外伤：通常属于意外险范畴而非疾病险

## 三、带病投保风险因子识别

### 3.1 时间节点异常分析

**高风险时间模式：**

```
投保日 ────────────── 确诊日

[<30天]   极高风险：投保目的明显
[30-90天] 高风险：等待期审查重点
[90-180天] 中高风险：需核查投保前就诊史
[>180天]   相对低风险：但仍需核查既往史
```

**需重点关注的时间节点证据：**
- 首次出现相关症状的时间（从病历中提取）
- 首次相关就诊时间（门诊、急诊）
- 首次检查出现异常指标的时间（体检报告）
- 投保健康告知日期
- 保单签发日期（保单生效日）
- 正式确诊日期

**如果：首次症状出现日期 < 保单生效日 → 强烈疑似带病投保**

### 3.2 既往病史隐瞒识别

**通过病历内容识别既往史：**

病历"既往史"记载通常包含被保险人的历史健康状况，重点关注：
- 既往史记载是否与健康告知书内容一致
- 入院记录中"诊断依据"栏的既往相关检查记录
- 用药史（尤其是慢性病用药）
- 过敏史（某些药物的使用提示相关疾病史）

**常见隐瞒病史的疾病与关联重疾对应关系：**

| 隐瞒的既往病史 | 可能申请的重疾 | 关联性评估 |
|--------------|--------------|----------|
| 高血压 | 急性心梗、脑中风 | 高度相关 |
| 糖尿病 | 尿毒症/肾衰竭、急性心梗 | 高度相关 |
| 乙肝/丙肝 | 肝硬化、肝癌 | 高度相关 |
| 肝硬化 | 肝癌、肝衰竭 | 高度相关 |
| 宫颈HPV感染 | 宫颈癌 | 相关 |
| 子宫肌瘤/卵巢囊肿 | 卵巢癌、子宫癌 | 关联存在但不直接 |
| 肺结节 | 肺癌 | 需评估结节性质和大小 |
| 肠息肉 | 结直肠癌 | 相关 |
| 慢性肾病 | 尿毒症 | 高度相关 |
| 冠心病 | 急性心梗、冠脉搭桥术 | 高度相关 |

## 四、保险欺诈类型识别指南

### 4.1 医疗造假欺诈

**A. 夸大病情（Minor-to-Major Inflation）**
- 特征：实际为轻症/良性疾病，但申请重大疾病理赔
- 案例模式：原位癌谎报为浸润癌；轻度肾损伤谎报为肾功能衰竭
- 识别方法：核查病理报告原件；核查肾功能指标实际数值

**B. 伪造/篡改医疗文书**
- 特征：修改病历关键内容（诊断名称、日期、指标数值）
- 案例模式：涂改病历日期至保单生效前；替换病理报告
- 识别方法：核查医院电子病历系统原始记录；病理号核查

**C. 假冒就医（Ghost Patient）**
- 特征：用他人病历申请理赔，或利用他人重病骗取自己的保险
- 识别方法：核查就医医院照片记录；向医院护士核实是否本人就诊

**D. 医患合谋**
- 特征：收买医生出具虚假诊断证明或病历
- 识别信号：医生非专科医师却出具复杂疾病诊断；多案件指向同一医院/医生
- 识别方法：向医院核查诊断医生执业资质；监控涉嫌医院的理赔频率

### 4.2 投保套现欺诈

**A. 密集投保后快速理赔**
- 特征：短期内在多家保险公司大额投保，随后申请重疾理赔
- 识别方法：通过行业信息共享平台核查多险情况；检查投保时间序列

**B. 家族性欺诈**
- 特征：家庭多名成员同期申请相似重疾理赔
- 识别方法：核查家族成员保险情况；多案件联动审查

**C. 代理人协助欺诈**
- 特征：保险代理人参与策划欺诈方案，协助客户伪造资料
- 识别信号：代理人与被保险人关系特殊（亲属、朋友）；代理人同时代理多个疑点案件

## 五、调查发起判断树

```
收到理赔申请
      │
      ▼
是否存在等待期/免责期问题？
  │ 是 → 直接认定不符合理赔条件，无需调查
  │ 否 → 继续
      │
      ▼
健康告知核查：投保前有无相关病史记录？
  │ 有明确记录且未申报 → 进入带病投保调查程序
  │ 无或不确定 → 继续
      │
      ▼
医疗资料真实性审查：是否存在疑点？
  │ 存在极高风险疑点（≥1个）→ 必须发起调查
  │ 存在高风险疑点（≥2个）→ 建议发起调查
  │ 存在中低风险疑点 → 书面核查后评估
  │ 无疑点 → 正常进入给付程序
      │
      ▼
保额是否超过敏感阈值（≥20万元）？
  │ 是 → 加强审核，必要时调查
  │ 否 → 评估疑点程度决定
```

## 六、调查内容标准化指引

### 6.1 向医院发起核查

**核查对象**：出具诊断的医院（病案室、相关科室）

**核查内容（优先级排序）**：
1. 被保险人的真实就诊记录（是否确实本人就医）
2. 病历记录的原始性（是否存在事后补录）
3. 检查报告的真实性（可要求医院出具核查意见）
4. 病理报告原件核查（病理号、制片日期、病理医师签名）
5. 主治医师信息确认（资质、是否认识患者或代理人）

**发起方式**：书面函件（加盖公司公章），注明被保险人信息、就医时间、请求核查事项

### 6.2 向社保/医保机构查询

**核查对象**：当地社会保险经办机构、医疗保险管理部门

**核查内容**：
- 被保险人在申请重疾前3-5年的医保就诊记录
- 是否有与本次重疾相关的历史就诊记录

**注意**：需取得被保险人授权，或通过法律程序调取

### 6.3 委托外部调查

**适用场景**：需要现场核查、走访调查的情形

**调查机构要求**：
- 具有合法资质的调查公司
- 签署保密协议
- 调查报告须加盖调查公司公章和调查员签名

**调查内容**：
- 被保险人目前真实健康状态（是否与病历描述一致）
- 就医行为核实（是否本人去医院就诊，就医时间是否与记录一致）
- 社会关系调查（代理人背景、医患关系）

### 6.4 行业信息共享查询

**查询平台**：中国保险行业协会理赔信息共享平台（具体名称以行业规定为准）

**查询内容**：
- 被保险人在其他保险公司的重疾理赔历史
- 短期内密集投保记录
- 是否被其他公司标记为高风险客户

## 七、特殊情形风险判断

### 7.1 职业风险判断

| 职业类型 | 风险提示 | 关注点 |
|---------|---------|--------|
| 医疗行业从业者 | 有条件获取或伪造医疗资料 | 病历真实性重点审查 |
| 保险从业者 | 熟悉理赔规则，可能设计方案 | 投保行为模式分析 |
| 律师/法务人员 | 擅长证据处理 | 投保告知合规性 |
| 无固定职业 | 动机分析 | 经济压力与保额匹配性 |

### 7.2 家庭经济状况与保额匹配性

**合理匹配原则**：重疾保额应与被保险人年收入相匹配（通常不超过年收入的10-15倍）

**异常信号**：
- 低收入群体大额投保（月收入<3000元但投保重疾保额>50万）
- 短期密集投保（1-2年内累积保额显著高于个人可承受范围）
- 多家公司同类险种叠加（累积重疾保额超过300万）
